# 解答編 L05〜L08（かっこ・小数・分数／比例式／個数と代金／過不足と速さ）

## L05 かっこ・小数・分数の方程式

### 練習

1. 3(x＋2)＝5x → 3x＋6＝5x → 6＝5x−3x → 2x＝6 → **x＝3**
   （検算: 左辺 3×(3＋2)＝15、右辺 5×3＝15 ✓）
2. 4(2x−1)＝5(x＋4) → 8x−4＝5x＋20 → 8x−5x＝20＋4 → 3x＝24 → **x＝8**
   （検算: 左辺 4×(16−1)＝60、右辺 5×(8＋4)＝60 ✓）
3. 0.7x−0.8＝0.5x＋0.6　両辺に10をかける: 7x−8＝5x＋6 → 2x＝14 → **x＝7**
   （検算: 左辺 0.7×7−0.8＝4.9−0.8＝4.1、右辺 0.5×7＋0.6＝3.5＋0.6＝4.1 ✓）
4. x/2−x/5＝3　両辺に10（分母2と5の最小公倍数）をかける: 5x−2x＝30 → 3x＝30 → **x＝10**
   （検算: 左辺 10/2−10/5＝5−2＝3、右辺 3 ✓）
5. (2x−1)/3＝(x＋4)/2　両辺に6をかける: 2(2x−1)＝3(x＋4) → 4x−2＝3x＋12 → **x＝14**
   （検算: 左辺 (2×14−1)/3＝27/3＝9、右辺 (14＋4)/2＝9 ✓）
6. まちがい: 両辺に5をかけて x−2＝15 とするところまでは正しいが、**そこで止まってしまい、最後の「−2を移項する（両辺に2を加える）」を行っていない**。x−2＝15 はまだ「x＝（数）」の形ではない——15は x−2 の値であって、xの値ではない。
   正しくは x−2＝15 → x＝15＋2 → **x＝17**（検算: 左辺 (17−2)/5＝15/5＝3、右辺 3 ✓）。
   誤答 x＝15 を元の式に代入すると、左辺 (15−2)/5＝13/5＝2.6 で右辺の3と合わず、検算で発覚する。

### stretch

**S1** 0.05x＋0.2＝0.1x−0.3　係数に0.05があるので両辺に**100**をかける: 5x＋20＝10x−30
→ 20＋30＝10x−5x → 5x＝50 → **x＝10**
（検算: 左辺 0.05×10＋0.2＝0.7、右辺 0.1×10−0.3＝0.7 ✓。10倍だと 0.5x＋2＝x−3 となり小数が残る——係数を見わたして倍率を選ぶ意味が分かる）

## L06 比例式——比の値で方程式に帰着

### 練習

1. x：8＝5：4 → 比の値で x/8＝5/4　両辺に8をかける: x＝40/4 → **x＝10**
   （検算: 10：8 の比の値は 10/8＝5/4。右の比の値と等しい ✓）
2. 9：x＝3：5 → 比の値で 9/x＝3/5　両辺にxをかける（比の後項は0でないので可）: 9＝(3/5)x　両辺に5/3をかける: **x＝15**
   （検算: 9：15 の比の値は 9/15＝3/5 ✓）
3. 白をxgとする。白：青＝5：2 だから x：30＝5：2 → x/30＝5/2　両辺に30をかける: x＝150/2 → **x＝75**
   答え: 白は**75g**（検算: 75：30 の比の値は 75/30＝5/2 ✓。白のほうが青より多く、5：2という場面の感覚とも合う）
4. 水をxmLとする。米：水＝5：6 だから 450：x＝5：6 → 450/x＝5/6　両辺にxをかける: 450＝(5/6)x　両辺に6/5をかける: **x＝540**
   答え: 水は**540mL**（検算: 450：540 の比の値は 450/540＝5/6 ✓。水は米よりやや多く、5：6の感覚と合う）

### stretch

**S1** 例1（x＝9）: 外側どうし 9×4＝36、内側どうし 12×3＝36 で**等しい**。
練習1（x＝10）: 外側 10×4＝40、内側 8×5＝40 で等しい。練習2（x＝15）: 外側 9×5＝45、内側 x×3＝15×3＝45 で等しい。
気づきの例: 「比例式では、外側どうしの積と内側どうしの積がいつも等しくなりそうだ」。
理由（例1で）: 比の値の等式 x/12＝3/4 の両辺に 12×4＝48 をかけると、左辺は 48×x/12＝4x（外側どうしの積）、右辺は 48×3/4＝36＝12×3（内側どうしの積）。つまりこの「積が等しい」は、比の値の等式の両辺に分母の積をかけただけのもの——等式の性質③から出てくる現象だと説明できる。

## L07 等しい2つの数量を見つける——個数と代金

### 練習

1. ① 鉛筆をx本とする。② 等しい2つの数量: 「買った物の代金の合計」と「実際の代金600円」。③ 左辺は買った物の代金の合計、右辺は実際の代金を表す。
   ④ 80x＋120＝600 → 80x＝480 → **x＝6**（検算: 左辺 80×6＋120＝600、右辺 600 ✓）
   ⑤ 答え: 鉛筆は**6本**（本数として自然な値で、場面に合う）。
2. ① x年後とする。② 等しい2つの数量: 「x年後の母の年齢」と「x年後のAさんの年齢の2倍」。③ 左辺はx年後の母の年齢、右辺はx年後のAさんの年齢の2倍を表す。
   ④ 42＋x＝2(14＋x) → 42＋x＝28＋2x → 42−28＝2x−x → **x＝14**（検算: 左辺 42＋14＝56、右辺 2×(14＋14)＝56 ✓）
   ⑤ 答え: **14年後**（そのときAさん28歳・母56歳。年齢として自然で、場面に合う）。
3. ① 50円切手をx枚とする（80円切手は 12−x 枚）。② 等しい2つの数量: 「買い方から計算した代金の合計」と「実際の代金750円」。③ 左辺は買い方から計算した代金の合計、右辺は実際の代金を表す。
   ④ 50x＋80(12−x)＝750 → 50x＋960−80x＝750 → −30x＝−210 → **x＝7**（検算: 左辺 50×7＋80×5＝350＋400＝750、右辺 750 ✓）
   ⑤ 答え: 50円切手は**7枚**（80円切手は5枚。枚数の合計 7＋5＝12枚 も場面に合う）。
4. 解答例: 「左辺 50x＋80(12−x) は**買い方から計算した代金の合計**を表し、右辺 750 は**実際に払った代金**を表す。この2つは**同じ1回の買い物の代金を二通りに表したもの**だから等しい。」

### stretch

**S1** りんごを x個とする（90円）。90x＋150(10−x)＝1200 → 90x＋1500−150x＝1200 → −60x＝−300 → **x＝5**
（検算: 左辺 90×5＋150×5＝450＋750＝1200、右辺 1200 ✓）
気づきの例: 「値段を入れかえても、答えはりんご5個のまま変わらない」。
説明の例: 元の例2の答えは「りんご5個・みかん5個」——**半々の買い方**だった。半々のときは、2つの値段を入れかえても代金の合計は 150×5＋90×5 のまま変わらない（かける相手が入れかわるだけ）。だから同じ合計10個・1200円という条件を、入れかえ後も半々の買い方が満たす。

## L08 過不足と速さ——1つの数量を二通りに表す

### 練習

1. 等しい2つの数量: 「5冊ずつ配ったときの総数の表し方 5x＋7」と「6冊ずつ配ったときの総数の表し方 6x−3」——**どちらもノートの総数**。
   生徒をx人として 5x＋7＝6x−3 → 7＋3＝6x−5x → **x＝10**
   （検算: 左辺 5×10＋7＝57、右辺 6×10−3＝57 ✓）
   答え: 生徒は**10人**、ノートは**57冊**（人数・冊数とも0以上の整数で場面に合う）。
   線分図: 全長57の線分を上下2段に——上段 5x（50）＋余り7、下段 6x（60）から不足3を切り欠く。
2. 姉が出発してからx分後に追いつくとする。等しい2つの数量: 「姉の道のり」と「妹の道のり」（追いつく瞬間、家からの道のりが等しい）。妹の歩いた時間は x＋6 分。
   80x＝50(x＋6) → 80x＝50x＋300 → 30x＝300 → **x＝10**
   （検算: 左辺 80×10＝800、右辺 50×(10＋6)＝800 ✓）
   答え: 姉は出発してから**10分後**に追いつく（家から800mの地点。時間として自然な値）。
3. x分後に出会うとする。等しい2つの数量: 「2人の道のりの**和**」と「2地点間の道のり1.8km＝**1800m**」（単位をmにそろえる）。
   70x＋50x＝1800 → 120x＝1800 → **x＝15**
   （検算: 左辺 70×15＋50×15＝1050＋750＝1800、右辺 1800 ✓）
   答え: **15分後**（Aさんは1050m、Bさんは750m歩いた地点で出会う）。
4. 解答例: 「左辺 5x＋7 も右辺 6x−3 も**ノートの総数**を表している。配り方（1人5冊か6冊か）がちがっても、**ノートの総数そのものは変わらない**から等しい。」

### stretch

**S1** 210x＝60(x＋9) → 210x＝60x＋540 → 150x＝540 → **x＝3.6**
3.6分＝3分＋0.6×60秒＝**3分36秒**後。
（検算: 左辺 210×3.6＝756、右辺 60×(3.6＋9)＝60×12.6＝756 ✓）
時間の長さは整数でなくてもよいので、この解はそのまま答えとして採用できる。「整数にならない解」の採否は数量の意味しだい——L09の主題につながる。
