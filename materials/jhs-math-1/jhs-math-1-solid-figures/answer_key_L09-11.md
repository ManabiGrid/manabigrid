# 解答編 L09〜L11

## L09 体積〜柱体と錐体

### 準備運動

1. 4×5×3＝**60（cm³）**
2. 12×5＝**60（cm³）**（底面積×高さ）
3. 体積の単位は**cm³**（立方センチメートル）。面積のcm²は「長さ×長さ」、体積のcm³は「長さ×長さ×長さ」で、次数が1つ多い。

### 練習（πの検算通過済み）

1. V＝15×6＝**90（cm³）**
2. V＝π×5²×8＝25π×8＝**200π（cm³）** ✓
3. V＝(1/3)×9×9×4＝(1/3)×324＝**108（cm³）**（81÷3＝27を先に→27×4＝108 ✓）
4. V＝(1/3)×π×6²×5＝(1/3)×180π＝**60π（cm³）**（36÷3＝12を先に→12×5＝60 ✓）
5. 誤り: 錐体なのに**1/2**を掛けている（平面の三角形＝長方形の1/2からの類推が混ざった形。立体では1/3）。正: V＝(1/3)×π×3²×4＝**12π（cm³）**
6. 円錐の体積＝同じ底面・高さの円柱の1/3 → 45π×1/3＝**15π（cm³）**

## L10 球の表面積と体積

### 練習（πの検算・r²/r³の見分け検算通過済み）

1. r＝6。V＝(4/3)×π×6³＝(4/3)×216π＝**288π（cm³）**（216÷3＝72→72×4＝288 ✓）。S＝4×π×6²＝**144π（cm²）** ✓
2. r＝2。V＝(4/3)×π×2³＝(4/3)×8π＝**32π/3（cm³）**（分数のままでよい。これが正確な値）。S＝4×π×2²＝**16π（cm²）** ✓
3. 直径10cm → 半径5cm。V＝(4/3)×π×5³＝(4/3)×125π＝**500π/3（cm³）** ✓
4. 半球 r＝3。
   (1) V＝球の半分＝(1/2)×(4/3)×π×3³＝(1/2)×36π＝**18π（cm³）** ✓
   (2) S＝曲面（球の表面積の半分）＋まっぷたつの面の平らな円＝(1/2)×4π×3²＋π×3²＝18π＋9π＝**27π（cm²）**（平らな円の面を忘れないこと＝「切ると表面が増える」） ✓
5. 「4×π×4²」は**表面積の式 S＝4πr²**の形（r²・係数4）。体積を問われているのだから r³・係数4/3 の式を使う。正: V＝(4/3)×π×4³＝(4/3)×64π＝**256π/3（cm³）** ✓
6. 円柱: V＝π×3²×6＝54π（cm³）。その2/3＝54π×2/3＝**36π（cm³）**。例題1の球の体積と**一致 ✓**

## L11 単元まとめ〜見る道具と測る道具の棚卸し

### 練習

1. 自己チェック: 本文の棚卸し表と突き合わせ、**書けなかった行のレッスンへ戻る**。表現は自分の言葉でよい（中身が合っていれば正解）。
2. 直方体ABCD-EFGH（上面ABCD・下面EFGH・AはEの真上）。
   (1) 辺DHとねじれの位置にある辺。消去の順で。交わる: DA・DC（点D）、HE・HG（点H）／平行: AE・BF・CG／残り＝**辺AB・BC・EF・FG**（検算: 12−1−4−3＝4本 ✓）。2条件チェック: いずれも①交わらない ②平行でない ✓

   (2) V＝π×4²×4＝**64π（cm³）**。S＝底面2枚＋側面＝2×16π＋（円周8π×高さ4）＝32π＋32π＝**64π（cm²）**（体積と表面積の数字がそろうのは1辺4cmの偶然——単位で区別 ✓）
4. 底面の円周＝2π×6＝12π。母線10cmの円全体の円周＝20π。12π/20π＝3/5 → 中心角＝360°×3/5＝**216°**。側面積＝π×10²×3/5＝**60π**。底面積＝36π。表面積＝60π＋36π＝**96π（cm²）**。検算: πrℓ＝π×6×10＝60π ✓

6. (1) 検出: **2条件チェック**（②平行でないか、以前に3分類の全体図）。「交わらない」だけでは平行ともねじれとも決まらない。辺ABと辺CGは①交わらない ②平行でない → 正しくは**ねじれの位置**。
