# 解答編 L09〜L12（文字と式・1節末＋2節）

## L09 不等式——大小関係を式にする

### 練習

1. (1) 「以上」＝境界を含む → **x≧40**（境界代入: x＝40 は「40人以上」に入る。≧ は 40 を含む ✓）
   (2) 代金 6a 円が 700円「より安い」＝境界を含まない → **6a＜700**（境界チェック: 代金がちょうど700円の場合は「700円より安い」に入らない → 含まない ＜ ✓）
   (3) 代金 2y＋150 円が 500円「以下」 → **2y＋150≦500**（境界チェック: ちょうど500円は「以下」に入る → 含む ≦ ✓）
   (4) 「未満」＝境界を含まない → **b＜20**（境界チェック: ちょうど20kgは「20kg未満」に入らない → 含まない ＜ ✓）
2. (1) **x は 15以下**（15 を含む）
   (2) **1個 a 円の品物5個の代金は、300円より高い**（「a の5倍は 300 より大きい」でもよい）
3. 境界の 12 を代入すると、x＜12 は「12 は入らない」と言っている。しかし「12冊以下」に 12冊は**入る**——不一致なので誤り。正しくは **x≦12**（境界を含む記号を選ぶ）。
4. (1) おつりの関係は「等しい」関係 → **等式 1000−a＝280**
   (2) 値段の大小関係 → **不等式 a＞1000**（「より高い」＝境界を含まない）

### stretch

**S1** 3≦x＜8 について——x＝3: 3≦3 は成り立つ（≦ は等しい場合を含む）ので**入る**。x＝5: 3≦5 も 5＜8 も成り立つので**入る**。x＝8: 8＜8 は成り立たない（＜ は等しい場合を含まない）ので**入らない**。左端は含み、右端は含まない範囲になっている。

## L10 項と係数——式を部品に分ける

### 練習

1. (1) **3x と 5**
   (2) **7 と −2x**（−は項の一部として切り取る）
   (3) **−x、4、−3x** の3つ
2. (1) **6**　(2) **−5**（符号ごと）　(3) **1**（x＝1×x のかくれた1）　(4) **−1**（−x＝(−1)×x）
3. (1) **4a と −7**
   (2) a＝1 → 4−7＝**−3**。a＝2 → 8−7＝**1**。a＝3 → 12−7＝**5**。動いているのは 4a の部分（4→8→12）で、**−7 はずっと変わらない**。文字の項＝動く部品、数の項＝動かない部品。
4. 誤り①: 項は「3x と 2」ではなく **−3x と 2**（符号ごと切り取る）。誤り②: 係数は「3」ではなく **−3**。符号を置き忘れると項も係数も別ものになる——2か所の誤りは、どちらも同じ「−の置き忘れ」から来ている。

### stretch

**S1** 3でわることは 1/3 をかけることと同じだから、x/3＝x×(1/3)＝**(1/3)×x**。よって係数は **1/3**。分数の形の項にも、ちゃんと係数がかくれている。

## L11 一次式の加法と減法

### 練習

1. (1) (5＋3)x＝**8x**　(2) (7−2)a＝**5a**　(3) (4−1)x＝**3x**（x の係数は1）　(4) (−2＋6)y＝**4y**
2. (1) 2x＋3x＋4＝**5x＋4**（検算: x＝2 → 元 4＋4＋6＝14、答 10＋4＝14 ✓）
   (2) 6a−2a−3＋8＝**4a＋5**（a＝2 → 元 12−3−4＋8＝13、答 8＋5＝13 ✓）
   (3) −x＋4x＋5−7＝**3x−2**（x＝2 → 元 5−2−7＋8＝4、答 6−2＝4 ✓）
3. (1) 3x＋2x＋2＋4＝**5x＋6**（x＝2 → 元 8＋8＝16、答 10＋6＝16 ✓）
   (2) 2x＋x−3＋1＝**3x−2**（x＝2 → 元 1＋3＝4、答 6−2＝4 ✓）
   (3) 5a−3a＋4−1＝**2a＋3**（a＝2 → 元 14−7＝7、答 4＋3＝7 ✓）
   (4) x−6−4x＋2＝**−3x−4**（−(4x−2) の 2 の符号反転に注意）
4. x＝2 のとき、元の式 (2−6)−(4×2−2)＝(−4)−6＝**−10**。答えの式 −3×2−4＝−6−4＝**−10**。一致 ✓
5. 誤りは −(2x−5) のかっこ外し。**−5 の符号も反転して ＋5** にしなければならないのに、−5 のままにしている。正しくは (6x＋1)−(2x−5)＝6x＋1−2x＋5＝**4x＋6**（検算: x＝1 → 元 7−(−3)＝10、答 4＋6＝10 ✓、誤答 4x−4 は 0 ✗）
6. 【説明の例】3x は「x 円の鉛筆 3本分」、2x は「x 円の鉛筆 2本分」。あわせると鉛筆は 3＋2＝5本で、代金は「x 円の 5本分」＝5x 円。つまり、**x が何個分あるかを数えて、個数（係数）どうしをたしている**——係数の式で書けば 3x＋2x＝(3＋2)x＝5x。x がどんな値段でも「3本分＋2本分＝5本分」は変わらないから、この計算はいつでも正しい。（検算の例: x＝100 → 元 300＋200＝500、答 5×100＝500 ✓）

### stretch

**S1** 2(n−2)＝(n−2)＋(n−2)＝n＋n−2−2＝**2n−4**。よって 2n＋2(n−2)＝2n＋2n−4＝**4n−4**——碁石の(ウ)の式は、計算すると(イ)の式と同じ形になる。

## L12 数×一次式と章のまとめ

### 練習

1. (1) 3×2x＋3×1＝**6x＋3**（検算: x＝2 → 元 3×5＝15、答 12＋3＝15 ✓）
   (2) −2×4x−2×(−3)＝**−8x＋6**（負×負＝正。x＝1 → 元 −2×1＝−2、答 −8＋6＝−2 ✓）
   (3) 8x/2−6/2＝**4x−3**（x＝2 → 元 10÷2＝5、答 8−3＝5 ✓）
   (4) 9x/(−3)＋3/(−3)＝**−3x−1**（x＝1 → 元 12÷(−3)＝−4、答 −3−1＝−4 ✓）
2. (1) 2x＋6＋6x−3＝**8x＋3**（x＝2 → 元 2×5＋3×3＝19、答 16＋3＝19 ✓）
   (2) 6x＋8−3x＋15＝**3x＋23**（−3×(−5)＝＋15。x＝1 → 元 2×7−3×(−4)＝14＋12＝26、答 3＋23＝26 ✓）
   (3) 8x−4−6x＋10＝**2x＋6**（x＝2 → 元 4×3−2×1＝10、答 4＋6＝10 ✓）
3. (1) りんご3個 3a 円と袋 50円 → **(3a＋50)円**
   (2) 3×120＋50＝360＋50＝**410**（a＝120 のとき代金は410円）
   (3) 「より安い」＝境界を含まない → **3a＋50＜500**
4. (2x＋7)＋3(x−4)＝2x＋7＋3x−12＝**5x−5**。x＝2 のとき、元の式 (4＋7)＋3×(2−4)＝11＋(−6)＝**5**、答えの式 10−5＝**5**。一致 ✓
5. 2n＋2(n−2)＝2n＋2n−4＝**4n−4**——同じ式になった。代入（n＝4、10）では「試した数で一致」までしか言えないが、計算は**文字 n のまま**式を変形している。n がどんな数を表していても成り立つ規則（分配・項をまとめる）だけを使ったから、「どんな n でも同じ」と言い切れる。

### stretch

**S1** 真ん中の整数を n とすると、3つの整数は n−1、n、n＋1。和は
(n−1)＋n＋(n＋1)＝n＋n＋n−1＋1＝**3n**
3n は 3×(整数) の形だから、連続する3つの整数の和は必ず**3の倍数**（真ん中の数の3倍）になる。（検算: 4＋5＋6＝15＝3×5 ✓、10＋11＋12＝33＝3×11 ✓——どちらも真ん中の数の3倍になっている）
