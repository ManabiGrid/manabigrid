# 解答編 L09〜L12

## L09 おうぎ形〜中心角に比例する

### 練習

1. 円全体（半径9cm）: 円周＝2π×9＝**18π (cm)**・面積＝π×9²＝**81π (cm²)**。
   中心角40° → 40/360＝**1/9**。弧の長さ＝18π×1/9＝**2π (cm)**・面積＝81π×1/9＝**9π (cm²)**。
   検算: 弧2πは円周18πの1/9 ✓・面積9πは81πの1/9 ✓
2. (1) 150°＝30°×5だから、30°の行を5倍して、弧＝π×5＝**5π (cm)**・面積＝3π×5＝**15π (cm²)**。
   検算（取り分方式）: 150/360＝5/12。12π×5/12＝5π ✓・36π×5/12＝15π ✓
   (2) 弧5πは円周12πの 5π/12π＝5/12。中心角＝360°×5/12＝**150°**。（(1)と同じおうぎ形になっている。順・逆の両向きで一致 ✓）
3. **面積も2倍になる**。理由の例:「同じ円からできるおうぎ形では、弧の長さも面積も**どちらも中心角に比例する**から、中心角が2倍（80°→160°）なら面積も2倍になる。」
4. まちがい: π×6²＝36πは**円全体**の面積。中心角60°の取り分 60/360＝1/6 をかけ忘れている。正しくは 36π×1/6＝**6π (cm²)**。防ぐ習慣:「式を書く前に、円全体の何分の何かを分数で言う」。

### stretch

**S1** 半径6cm・中心角60°: 弧＝12π×(60/360)＝12π×1/6＝**2π (cm)**。
半径3cm・中心角240°: 円周6π、240/360＝2/3、弧＝6π×2/3＝**4π (cm)**。
中心角は4倍（60°→240°）だが、弧は2π→4πの**2倍**にしかなっていない。→ 弧の長さが中心角に比例するのは「**同じ半径どうしで比べたとき**」だけ。半径が変われば円全体の円周（比例の出発点）が変わるから、中心角の倍率だけでは決まらない。

## L10 おうぎ形の弧の長さと面積

### 練習

1. (1) 半径3cm・240° → 240/360＝**2/3**。弧＝2π×3×2/3＝6π×2/3＝**4π (cm)**・面積＝π×9×2/3＝**6π (cm²)**。
   検算: 半円（180°）なら弧3π・面積4.5π。2/3は1/2より大きいから、それより大きい4π・6πでつじつま ✓
   (2) 半径10cm・54° → 54/360＝**3/20**。弧＝20π×3/20＝**3π (cm)**・面積＝100π×3/20＝**15π (cm²)**。
   検算: 3/20は1/4（90°）より小さい取り分。90°なら弧5π・面積25π。それより小さい ✓
   (3) 半径12cm・150° → 150/360＝**5/12**。弧＝24π×5/12＝**10π (cm)**・面積＝144π×5/12＝**60π (cm²)**。
   検算: 半円なら弧12π・面積72π。5/12は1/2より少し小さい。10π・60πはそれより少し小さい ✓
2. 中心角a°として、弧4πは円周10πの 4π/10π＝**2/5**。a＝360×2/5＝**144°**。
   順向き検算: 半径5cm・144° → 144/360＝2/5。弧＝10π×2/5＝4π ✓
3. 面積60πは円全体144πの 60π/144π＝**5/12**。中心角＝360°×5/12＝**150°**。
   弧の長さ＝24π×5/12＝**10π (cm)**。
   順向き検算: 半径12cm・150°の面積＝144π×5/12＝60π ✓（L10練習1(3)と同じおうぎ形）
4. しかけたまちがい:「S＝π×6²＝36π」で**止まりかけた**。円全体の面積を、おうぎ形の面積として答えかけた（取り分のかけ忘れ）。防ぐ習慣の例:「**式を書く前に**、円全体の何分の何か（120/360＝1/3）を先に書く」。先に1/3と書いてあれば、36πで手が止まらず×1/3まで一息に進める。（この答案のように途中で気づいて検算までできれば、それも立派な運用だ）

### stretch

**S1** 導出: S＝πr²×(a/360)＝(1/2)×r×**2πr×(a/360)**＝(1/2)×r×ℓ＝**(1/2)ℓr**。
（2πr×(a/360)の部分がℓそのものであることに気づくのがポイント。πr²＝(1/2)×r×2πr の変形は、r²＝(1/2)×r×2r による）
照合: 練習1(1)は ℓ＝4π・r＝3 → S＝(1/2)×4π×3＝6π。取り分方式の答え6πと一致 ✓
イメージ: おうぎ形を中心から細かく切ると、1切れは「底辺＝弧の切れはし・高さ＝r」の細い三角形に近い。底辺の合計がℓだから、面積の合計は(1/2)×ℓ×rに近づく——三角形の面積公式と同じ構造が現れる。（おなじみの「交互に並べ替える」図で考えるなら、底辺がおよそℓ/2・高さがrの平行四辺形に近づき、(ℓ/2)×r＝(1/2)ℓrで同じ答えになる）

## L11 円の接線

### 練習

1. (3)の解答例:「接線は接点Aを通る半径OAに垂直である【根拠: 円の接線は接点を通る半径に垂直】。だから、直線OA上の点Aを通るOAの垂線を作図すれば、それがAにおける接線である【根拠: 垂線の作図（L05・直線上の点を通る場合）】。」
2. 気づきの例:「**PA＝PBになっている**（コンパスで写し取るとぴったり重なる）。」——円の外の点Pから2つの接点までの長さは等しい。理由づけは中3で（ここでは観察の記録でよい）。
3. まちがい:「どの半径とも垂直」。接線が垂直なのは**接点を通る半径**だけ。正しい文の例:「円の接線は、**その接点を通る半径**に垂直である。」（接点以外の点を通る半径とは、垂直に交わらない——図をかけば一目でわかる）
4. 距離＝**3cm（半径と等しい）**。理由の例:「中心Oと直線ℓの距離はOからℓへの垂線の長さで、接線⊥半径OAより、その垂線が半径OAそのものだから【根拠: 接線は接点を通る半径に垂直・点と直線の距離は垂線の長さ】。」

### stretch

**S1** 作図の要点: 3点それぞれで「半直線OA・OB・OCをのばす→各点を通る垂線」を3回。3点を円周のほぼ3等分の位置など大きく離してとり、3本の接線が**円Oを囲む三角形**をつくれば、円Oはその3辺すべてに内側から接する円になっている（3点が円周の一部にかたよっていると、接線どうしの三角形が円の外側にできて「3辺に内側から接する」にならない配置も起こる——問題文の「大きく離した3点」はそのための条件だ）。調べ方の例:「三角形 すべての辺に接する円」——先の学年で名前付きで再会する。

## L12 単元まとめ〜作図と円の総合

### 総合演習

1. 条件1を満たす点の集まり＝**線分ABの垂直二等分線**【根拠: 2点から等距離の点の集まり】。手順:「①線分ABの垂直二等分線を作図する ②その直線と直線ℓとの**交点**をPとする【根拠: Pは①の線上にあるからPA＝PB、かつℓ上にある——両条件を満たす】。」
2. 作図の要点: 各頂点について「頂点とOを通る**直線**をひく→コンパスでOから頂点までの長さを写し取り、**Oをはさんで反対側**に対応点をとる」を3回。説明の例:「△DEFは、△ABCを、**点Oを中心に180°**回転移動（点対称移動）した図形である。」（180°だから向きの記載は省略できる——L03練習4）
3. 中心角270° → 270/360＝**3/4**。円全体（半径6cm）: 円周12π・面積36π。
   弧の長さ＝12π×3/4＝**9π (cm)**・面積＝36π×3/4＝**27π (cm²)**。
   まわりの長さ＝弧＋半径2本＝9π＋6＋6＝**9π＋12 (cm)**。
   検算: 切り取った90°側（弧3π・面積9π）と合わせると、弧9π＋3π＝12π＝円周 ✓・面積27π＋9π＝36π＝円全体 ✓
4. (1) 90/360＝**1/4**。円全体（半径4cm）: 円周8π・面積16π。弧＝8π×1/4＝**2π (cm)**・面積＝16π×1/4＝**4π (cm²)**。
   検算: 4分の1円。円周8πの1/4＝2π ✓
   (2) 予想:「**交わらない**（どこまでのばしても出会わない。平行に見える）」。観察の理由の例:「接線はOAに垂直（接線上にとった点Pで∠OAP＝90°）で、OBもOAに垂直（∠AOB＝90°）。同じ直線OAに垂直な2本の直線は、たがいに平行になっているように見える。」——「垂直⇒平行」をきちんと根拠から導くのは中2「平行線」の仕事。ここでは図からの予想と観察が書けていれば十分。
