# L10 章末まとめ——文字の力の総点検

## ねらい

- この章で身につけた5つの力（計算する・表す・説明する・変形する・読む）を、1枚の地図として整理する。
- 総合演習で自分の弱点部品を特定し、次の章（連立方程式）へ進む準備を整える。

## この章の地図

この章でやってきたことは、5つの部屋に整理できる。

| 部屋 | できるようになったこと | 主なレッスン |
|---|---|---|
| 計算する | 同類項をまとめる・かっこを外す・単項式どうしの乗除 | L01〜L04 |
| 表す | 偶数2n・奇数2n＋1・2けたの数10a＋b（事象を式に） | L05 |
| 説明する | 表す→計算・変形→読む→結論の4ステップ | L06〜L07 |
| 変形する | 目的の文字について解く（a＝2S/h） | L08 |
| 読む | 結果の式から事実を取り出す（差はいつでも2π） | L09 |

見取り図はこうだ。**「計算する」は他の4部屋すべての土台**——表しただけでは終われず（L05）、説明の中心には変形があり（L06）、読むためには整理された式が要る（L09）。逆に言えば、計算練習の意味は、この4部屋で使うためにあった。

自己チェックの目安: 各部屋について「代表問題を1問、人に説明するつもりで、声に出して手順を言いながら解ける」なら合格。手が止まった部屋が、テスト前に戻るべき場所だ。

:::guide
**間違い探しは「型」で戻る**

総合演習で間違えたら、この章で立てた型のどれが抜けたかを探すと復習が速い。①符号ごと項に分ける（L01）②文字の部分は同じ？（L02）③−は全員に配る（L03）④÷のうしろはまるごと分母（L04）⑤式は代入でテスト（L05）⑥読みの一文を書いたか（L06）⑦両辺全体をわったか（L08）。間違いの多くは、この7つのどれかの省略で起きる。
:::

## 総合演習

1. 次を計算しよう。
   (1) 3(2x−y)−(x−4y)　(2) (−6a²b)÷3ab×(−2b)
2. a＝2、b＝−3 のとき、12ab²÷6b の値を求めよう（先に式を簡単にすること）。
3. 「連続する2つの奇数の和は、4の倍数になる」を4ステップで説明しよう。（表す: 連続する2つの奇数は 2n＋1、2n＋3。読みたい形: 4×（整数）。）
4. 台形の面積の公式 S＝(1/2)(a＋b)h を h について解こう。また、その式が「どんな場面で便利か」を一言で添えよう。
5. n を整数とするとき、7n＋7 はどんな数か、式の形を根拠にして読み取ろう。

:::zatsudan
この章の道具は、このあと引っぱりだこになる。連立方程式では 2(3x−2y)−3(2x＋5y) の型の計算で文字を消し、一次関数では「y について解く」変形が毎回の準備運動になり、図形の証明では「4ステップで説明する」書き方がそのまま論証の書き方につながる。そして中3では、かけ算の相手が単項式から多項式へ広がる（式の展開）。つまりこの章は終わりじゃなくて、2年生の数学ぜんぶの**開幕戦**だったというわけ。
:::

## 次の章への接続

- **連立方程式**……係数をそろえて文字を消す＝L03の複合計算がそのまま主役。
- **一次関数**……y＝の形に直してから調べる＝L08の「y について解く」。
- **図形の性質と証明**……根拠を添えて結論を導く記述＝L06〜L07の4ステップの図形版。
- **中3・式の展開と因数分解**……単項式×多項式・多項式×多項式へ。L04の「どうしまで」の柵が外れる日。

:::stretch
**S1** この章の総仕上げとして、自分で「整数の性質」をひとつ予想し、4ステップで説明してみよう。（例のつくり方: 連続する4つの整数の和は？ 奇数と奇数の差は？　まず具体的な数で3例実験し、予想を立ててから文字で確かめる。予想が外れたら、それは反例が見つかったということ——それも立派な数学の成果だ。）
:::
