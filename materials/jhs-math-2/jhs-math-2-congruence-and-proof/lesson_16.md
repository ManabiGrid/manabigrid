# L16 章末まとめ〜根拠のリストを棚卸しする

## ねらい

- この章で「証明の根拠に使ってよいこと」になった事柄を**一覧に棚卸し**し、それぞれが「認めた出発点」か「導いた定理」かを区別できるようにする。
- 総合演習で、リストから根拠を選んで使う練習をする。

## 主概念：根拠のリスト〜この章であなたが手に入れたもの

L01の最初の根拠は「一直線の角は180°」の1つだけだった。16レッスンかけて、リストはここまで育った。**「認めた」と「導いた」の列**に注目してほしい——導いたものは、すべて左の列のどれか（とそれ以前に導いたもの）から証明されている。

| 根拠 | 認めた/導いた | どこで |
|---|---|---|
| 一直線の角は180° | 認めた（出発点） | L01 |
| 対頂角は等しい | 導いた | L01 |
| 平行線の性質（同位角相等） | 認めた（出発点） | L02 |
| 平行線の性質（錯角相等） | 導いた | L02 |
| 平行線になるための条件（同位角相等→平行） | 認めた（出発点） | L02 |
| 平行線になるための条件（錯角相等→平行） | 導いた | L02 |
| 同側内角の和は180°／和が180°なら平行 | 導いた | L02（練習4・S1） |
| 三角形の内角の和は180° | 導いた | L03 |
| 三角形の外角＝となり合わない2内角の和 | 導いた | L03 |
| n角形の内角の和180°×(n−2)・外角の和360° | 導いた | L04 |
| 合同な図形の性質（対応する辺・角は等しい） | 定義から | L05 |
| 三角形の合同条件（3つ） | 認めた（出発点） | L06 |
| 直角三角形の合同条件（2つ） | **導いた** | L10 |
| 二等辺三角形の性質（底角・頂角の二等分線） | 導いた | L08 |
| 二等辺三角形になるための条件（2角相等→二等辺） | 導いた | L09 |
| 正三角形の性質・条件 | 導いた | L09 |
| 平行四辺形の性質（3つ） | 導いた | L11 |
| 平行四辺形になるための条件（5つ） | 導いた（1は定義） | L12 |
| 長方形・ひし形・正方形の対角線の性質 | 導いた | L13 |

出発点として「認めた」のは、実は**一直線の角・平行線の性質と条件・三角形の合同条件**の数件だけ。残りは全部、そこから証明の連鎖で導いた。二等辺三角形も平行四辺形も、あなたはもう「教わったから知っている」のではなく、**「なぜ成り立つか、出発点までさかのぼって言える」**。この違いが、この章の到達点だ。

あわせて、道具の使い方の型も棚卸ししておこう。

- **仮定と結論の抜き出し**を設問1に（白紙で止まらない・L07）
- **方針メモ**: 仮定/結論/使えそうな根拠の3点（L07）
- **見直し3点チェック**: ①結論を根拠にしない ②根拠の文言は実在するか ③結論まで渡りきったか（L07）
- **逆を作ったら新規に点検・正しくないなら反例1つ**（L14）
- **証明は読み返すと、追加の性質が掘り出せる**（L15）

:::guide
**リストの使い方〜「もらった地図」ではなく「自分の足あと」として**

上の表は、テスト前の総点検にそのまま使える。おすすめは、表の「導いた」の行を1つ選んで、**何も見ずに証明を再現してみる**こと（L01練習4でやった、あの再現練習の章末版だ）。再現できれば、その定理は根拠のリストの「生きた道具」になっている。再現に詰まったら、該当レッスンの方針メモだけ見て、もう一度。丸暗記ではなく「方針から組み立て直せる」が目標だ。
:::

:::guide
**まちがえた問題の記録は「どのチェックで落ちたか」まで**

総合演習や過去の練習でまちがえたら、「×」だけで終わらせず、**見直し3点チェックのどれで落ちたか**（①循環／②文言・隠れた前提／③渡りきり不足）、あるいはチェック以前（方針メモが立たなかった・仮定の翻訳ミス）かを一言記録しよう。まちがいの型が分かれば、対策するレッスンが特定できる（①→L08練習4、②→L06 guide、③→L12練習5）。この章の教材は、型ごとに戻る場所が用意してある。
:::

:::zatsudan
この章の根拠のリスト、実はこれで完成ではない。中学3年の図形では、「形が同じで大きさが違う」図形たち（L06の雑談で少し顔を出した、あの拡大コピーの関係）を調べることになっていて、そこで新しい道具がこのリストに加わる。リストが増えれば、証明できることも増える。今日の棚卸しは店じまいではなくて、次の仕入れの前の在庫確認、というわけだ。
:::

## 総合演習

1. ▱ABCDの辺BC上に、Cとは異なる点Eを、DC＝DEとなるようにとる。∠DEC＝∠BADであることを証明しよう（方針メモ: ∠DECは二等辺三角形DECの底角。∠DCEと∠BADの関係は？　平行四辺形の対角）。
2. ▱ABCDの辺ADの中点をM、辺BCの中点をNとする。四角形MBNDが平行四辺形になることを証明しよう（方針メモ: MDとBNに注目。平行はただでもらえる。長さは「半分どうし」。どの性質で元が等しいと言える？　条件5）。
3. 次の命題の逆を作り、正しいかどうか判定しよう（正しくないなら反例を1つ）。
   「四角形ABCDがひし形ならば、四角形ABCDの4つの辺はすべて等しい。」
4. 【読む】次の答案を見直し3点チェックにかけよう。
   「▱ABCDで、対角線ACを引く。△ABCと△CDAで、AB＝CD（平行四辺形の対辺は等しいから）、BC＝DA（同）、AC＝CA（共通）。対応する3組の辺がそれぞれ等しいから△ABC≡△CDA。よって平行四辺形の対辺は等しい ■」

:::stretch
**S1** この章の「認めた出発点」のうち、「平行線になるための条件（同位角が等しければ平行）」だけを使って、「平行線の性質（平行ならば同位角が等しい）」を導くことはできるだろうか——逆どうしの関係にある2つを、片方だけ認めてもう片方を出せるか、という問いだ。少し考えてから、「もし錯角が等しくなかったら、どうなるはずか」という筋（背理法と呼ばれる、高校で正式に学ぶ考え方の入り口）を調べてみたい人は、「平行線の性質 証明 背理法」で調べてみよう。
:::
