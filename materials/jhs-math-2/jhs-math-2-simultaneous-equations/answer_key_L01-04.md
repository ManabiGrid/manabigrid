# 解答編 L01〜L04（解の意味・加減法・代入法）

## L01 条件が2つあるとき——二元一次方程式と連立方程式

### 練習

1. 各組を代入して確かめる。
   ア (1, 7): 3×1＋7＝10 ○　イ (2, 3): 3×2＋3＝9 ×　ウ (3, 1): 3×3＋1＝10 ○　エ (4, −2): 3×4＋(−2)＝10 ○
   解は**ア・ウ・エ**（負の数の組も、式さえ成り立てば解であることに注意）。
2. x＝10−2y にyの値を順に入れる。y＝1→x＝8、y＝2→x＝6、y＝3→x＝4、y＝4→x＝2（y＝5だとx＝0で自然数でない）。
   **(2, 4), (4, 3), (6, 2), (8, 1) の4組**（検算例: 2＋2×4＝10 ✓）。
3. x＋y＝6 の自然数解: (1, 5), (2, 4), (3, 3), (4, 2), (5, 1)
   2x＋y＝9 の自然数解: (1, 7), (2, 5), (3, 3), (4, 1)
   共通する組は **(3, 3)**。検算: 3＋3＝6 ✓、2×3＋3＝9 ✓（両方成り立つので解）。
4. (1) **×**。解は1組ではない——(1, 7), (2, 6)など、**一つとは限らない**（自然数に限っても7組、範囲を広げればいくらでもある）。
   (2) **×**。「どちらか一方」ではなく**二つの方程式を同時に**（両方とも）成り立たせる組が連立方程式の解。

### stretch

**S1** xを0以上の整数に広げると (0, 7) が加わって**4組**（0＋7＝7…2×0＋7＝7 ✓）。整数全体に広げると、x＝4→y＝−1（2×4−1＝7 ✓）、x＝5→y＝−3、x＝−1→y＝9…と**いくらでも作れる（組は無数）**。気づきの例: 「解がいくつあるかは、式だけでなく**変域**で決まる」。

## L02 文字を消すという作戦——等式の性質と加減法

### 準備運動（前提診断）

1. 3x−5＝7 → 3x＝12 → **x＝4**（検算: 12−5＝7 ✓）
2. 4x＋6＝x−3 → 4x−x＝−3−6 → 3x＝−9 → **x＝−3**（検算: 左辺 −12＋6＝−6、右辺 −3−3＝−6 ✓）
3. (3x＋2y)＋(2x−2y)＝**5x**（yが消える——今日の授業の伏線）
4. 2(3x−2y)−3(2x＋5y)＝6x−4y−6x−15y＝**−19y**（xが消える。かっこを外すときの符号反転に注意）

### 練習

1. (1) ①＋②: 2x＝14 → x＝7、①に代入 7＋y＝9 → y＝2。**(7, 2)**（検算: 7＋2＝9 ✓、7−2＝5 ✓）
   (2) ①−②: x＝13−9＝4、②に代入 4＋y＝9 → y＝5。**(4, 5)**（検算: 8＋5＝13 ✓、4＋5＝9 ✓）
   (3) ①−②: (3x＋2y)−(3x−y)＝3y、19−13＝6 → 3y＝6 → y＝2、②に代入 3x−2＝13 → x＝5。**(5, 2)**（検算: 15＋4＝19 ✓、15−2＝13 ✓）
   (4) ①−②: 2x＝16−8＝8 → x＝4、②に代入 12＋2y＝8 → 2y＝−4 → y＝−2。**(4, −2)**（検算: 20−4＝16 ✓、12−4＝8 ✓。解が負の数でも構わない）
2. まちがい: ①−②の**右辺を計算していない**（左辺だけ引いて y＝10 としている）。右辺も 10−6＝4 を引く。
   正しくは ①−②: y＝**4**、②に代入 x＋4＝6 → x＝2。**(2, 4)**（検算: 2＋8＝10 ✓、2＋4＝6 ✓）
3. 予想: 加えると 3y＋(−3y)＝0 で**yが消える**。引くと 4x−4x＝0 で**xが消える**。
   加える場合: 8x＝16 → x＝2、①に代入 8＋3y＝11 → y＝1。引く場合: 6y＝6 → y＝1 → x＝2。どちらでも **(2, 1)**（検算: 8＋3＝11 ✓、8−3＝5 ✓）。

### stretch

**S1** ①A＝B の両辺に**同じ数C**を足しても等式は成り立つ（等式の性質）から、A＋C＝B＋C。ここで②C＝Dより、右辺のCはDに置きかえられる（同じものだから）。よって **A＋C＝B＋D**。——「辺々を加える」は、等式の性質（両辺に同じ数を足す）と「等しいものは置きかえられる」の合わせ技だと分かる。

## L03 係数をそろえる——加減法の完成

### 練習

1. (1) 宣言例: 「xを消す。①×2で係数2にそろえて引く」。①×2: 2x＋6y＝22。①×2−②: y＝22−19＝3、①に代入 x＋9＝11 → x＝2。**(2, 3)**（検算: 2＋9＝11 ✓、4＋15＝19 ✓）
   (2) 宣言例: 「yを消す。②×3で±3yにそろえて加える」。②×3: 6x＋3y＝12。①＋②×3: 10x＝30 → x＝3、②に代入 6＋y＝4 → y＝−2。**(3, −2)**（検算: 12＋6＝18 ✓、6−2＝4 ✓）
   (3) 宣言例: 「xを消す。係数を6にそろえて引く」。①×3: 6x＋9y＝15、②×2: 6x＋8y＝14。引いて y＝1、①に代入 2x＋3＝5 → x＝1。**(1, 1)**（検算: 2＋3＝5 ✓、3＋4＝7 ✓）
   (4) 宣言例: 「yを消す。①×2で∓4yにそろえて加える」。①×2: 10x−4y＝8。②＋①×2: 13x＝26 → x＝2、②に代入 6＋4y＝18 → y＝3。**(2, 3)**（検算: 10−6＝4 ✓、6＋12＝18 ✓）
2. 加える派: ①＋②: 6x＝18 → x＝3、②に代入 6−3y＝0 → y＝2。
   2倍して引く派: ②×2: 4x−6y＝0。①−②×2: 9y＝18 → y＝2 → x＝3。
   どちらも **(3, 2)**（検算: 12＋6＝18 ✓、6−6＝0 ✓）。異符号でそろっている文字（±3y）は「足すだけ」で消せるので、今回は加える方が速い。
3. まちがい: ②を3倍するとき**右辺に3をかけていない**（5のまま。正しくは15）。
   正しくは ②×3: 3x＋6y＝**15**。①−②×3: −4y＝7−15＝−8 → y＝2、②に代入 x＋4＝5 → x＝1。**(1, 2)**（検算: 3＋4＝7 ✓、1＋4＝5 ✓）
   （「解が分数になって変だ」という違和感から誤りに気づけたのは良い姿勢——検算とセットで武器になる。ただし**分数の解をもつ連立方程式もふつうにある**ので、「分数＝まちがい」と覚えないこと。違和感はあくまで検算を始めるきっかけ）

### stretch

**S1** (ア) xを消す: ①×3: 6x＋9y＝36、②×2: 6x＋4y＝26。引いて 5y＝10 → y＝2、①に代入 2x＋6＝12 → x＝3。
(イ) yを消す: ①×2: 4x＋6y＝24、②×3: 9x＋6y＝39。引いて 5x＝15 → x＝3 → y＝2。
どちらも **(3, 2)**（検算: 6＋6＝12 ✓、9＋4＝13 ✓）。
①＋②: 5x＋5y＝25 → 5でわって x＋y＝5。{ 2x＋3y＝12, x＋y＝5 } を解くと、②'×2: 2x＋2y＝10、①−②'×2: y＝2 → x＝3——**同じ解になる**。気づきの例: 「辺々を加えてできた式も、元の解を必ず満たす。しかも元の式を1本残しておけば、新しい式から残した式を引き算すると外した式が復元できる——**行きも帰りもできる**から、1本を残して組みかえるかぎり解は変わらない」。（元の2本を両方とも捨てて和の式だけにすると、帰り道がなくなって解が絞れなくなる——「1本残す」が効いている）

## L04 代入法と解法の選択

### 練習

1. (1) ①を②に代入: x＋3x＝8 → 4x＝8 → x＝2、①より y＝6。**(2, 6)**（検算: 6＝3×2 ✓、2＋6＝8 ✓）
   (2) ①を②に代入: 2(2y−1)＋3y＝12 → 4y−2＋3y＝12 → 7y＝14 → y＝2、①より x＝2×2−1＝3。**(3, 2)**（検算: 3＝4−1 ✓、6＋6＝12 ✓）
   (3) 右辺どうしを結ぶ: x−2＝−2x＋7 → 3x＝9 → x＝3、①より y＝1。**(3, 1)**（検算: 1＝3−2 ✓、1＝−6＋7 ✓）
2. (ア) 代入法: ②を①に代入 2x＋(x＋1)＝10 → 3x＝9 → x＝3、②より y＝4。
   (イ) 加減法: ②を移項して −x＋y＝1。①−②': (2x＋y)−(−x＋y)＝3x、10−1＝9 → 3x＝9 → x＝3 → y＝4。
   どちらも **(3, 4)**（検算: 6＋4＝10 ✓、4＝3＋1 ✓）。
3. (1) 宣言例: 「y＝…の形があるので**代入法**」。①を②に代入: 3x＋4x−3＝11 → 7x＝14 → x＝2、①より y＝5。**(2, 5)**（検算: 5＝8−3 ✓、6＋5＝11 ✓）
   (2) 宣言例: 「±2yで係数がそろっているので**加減法**（加える）」。①＋②: 8x＝8 → x＝1、①に代入 3＋2y＝1 → y＝−1。**(1, −1)**（検算: 3−2＝1 ✓、5＋2＝7 ✓）
4. 2(3y−5)−y＝5 → 6y−10−y＝5 → 5y＝15 → y＝3、x＝3×3−5＝4。**(4, 3)**（検算: 4＝9−5 ✓、8−3＝5 ✓。かっこを忘れて 2×3y−5−y と書くと −10 が −5 に化けるのが定番事故）

### stretch

**S1** (ア) 加減法: ②×3: 9x−3y＝6。①＋②×3: 11x＝22 → x＝2、②に代入 6−y＝2 → y＝4。
(イ) 代入法: ②を変形 y＝3x−2。①に代入: 2x＋3(3x−2)＝16 → 11x−6＝16 → x＝2 → y＝4。
どちらも **(2, 4)**（検算: 4＋12＝16 ✓、6−4＝2 ✓）。手数はほぼ互角——「−yの式は y＝…に直しやすいので代入法も速い」等、理由が言えればどちらを選んでもよい。
