## L06 「〜でない」確率——1から引くという近道

### 練習

1. ①直接: 6の目が出ないのは1〜5の**5通り** → **5/6**
   ②1から引く: 1−（6の目が出る確率1/6）＝**5/6**。**一致** ✓
2. 「少なくとも1枚は裏」が起こらないのは「2枚とも表」（1/4）のときだけ。
   1−1/4＝**3/4**（検算・直接数え: 4通り中（表裏）（裏表）（裏裏）の3通り → 3/4 ✓）
3. 同じ目になるのは6通り（6/36＝1/6）だから、同じにならない確率は 1−6/36＝30/36＝**5/6**（検算・直接数え: 36−6＝30マス → 30/36 ✓）
4. 絵札はジャック・クイーン・キングが4種類の組（マーク）ごとに1枚ずつで 3×4＝**12枚**。絵札を引く確率は12/52だから、絵札でない確率は 1−12/52＝40/52＝**10/13**（検算: 絵札でないカードは52−12＝40枚 → 40/52 ✓）

## L07 くじ引きは何番目でも公平か

### 練習

1. 6通り（L07主概念2の樹形図）のうち、AもBも当たらないのは（は1,は2）（は2,は1）の**2通り** → 2/6＝**1/3**（検算: Aが当たる2＋Bが当たる2＋どちらも当たらない2＝6 ✓）
2. 戻す場合、Bが引くときも3本すべてが袋にある。Bの引き方は3通り（同様に確からしい）で当たりは1本 → **1/3**。**戻さない場合（1/3）と同じ**。（ルールを変えてもBの当たりやすさは変わらない、という点まで書ければ完答。）
3. くじは当・はの2本。引き方は（当,は）（は,当）の**2通り**（同様に確からしい）。
   ①起こり得る場合は全部で2通りで、どれも同様に確からしい。
   ②Aの当たる確率は（当,は）の1通りで**1/2**、Bの当たる確率は（は,当）の1通りで**1/2**。
   ③確率が等しいから、**このくじ引きは公平である**。
4. （反論の例）樹形図で全部の場合をかき切ると6通りあり、どれも同様に確からしい。先に引くAの当たる確率は2/6＝1/3、後に引くBの当たる確率も2/6＝1/3で**等しい**。つまり引く順番によって当たりやすさは変わらないので、「最後に引く人が有利」という主張は**正しくない**。（「残り物に福」は確率の根拠を持たないことばである、という指摘まで書ければ十分。）

### stretch

**S1** 3本のくじ（当・は1・は2）をA→B→Cが引き切るので、場合は3人へのくじの配られ方＝**6通り**（当がAの並び2通り・Bの並び2通り・Cの並び2通り）。Cが当たるのは当が3番目の（は1,は2,当）（は2,は1,当）の**2通り** → 2/6＝**1/3**。Aも1/3、Bも1/3（当が1番目・2番目の場合がそれぞれ2通り）——**3人とも1/3で、最後の人も不利ではない**。

4. 4通り（樹形図）のうち、あなたの勝ち＝（表表）の1通りで**1/4**、わたしの勝ち＝（裏裏）の1通りで**1/4**。勝つ確率が等しいから、**公平と言える**（1枚ずつの2通り＝1/2は引き分けで、どちらの有利にもならない）。
