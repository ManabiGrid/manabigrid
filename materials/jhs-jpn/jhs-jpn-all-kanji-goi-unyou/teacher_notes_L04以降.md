# 教師用ノート（Lesson 04〜07）

## 1. L04〜L07の位置づけ（形式勾配の後半）

L1〜L3が「手続きの習得（選択中心〜部分書字）」、L4〜L7は「形式勾配を上げた運用」。設問形式が学習行動を規定するため、後半は部分書字→完全書字→産出（校正・条件作文）へと意図的に負荷を上げている。書字式では無解答が多い傾向があり、内部調査資料はこれを回避行動と**推論**している（無解答率だけでは「想起できなかった」「時間が足りなかった」等と区別できないため、回避と断定しない——内部調査資料 §10の推論表示を保持。無解答率の具体数値は調査名・年度の照合が本ノートで未完のため記載しない——年度なし数値の使用禁止原則による）。対処として、各活動の冒頭問は既習の手順をなぞれば書ける難度に置き、L4問10・L7問7のように「手順の言語化」自体を得点対象にして、空欄でも手続き点が取れる設計にしてある。 〔根拠区分: Derived〕

## 2. 誤概念対応の配置（内部調査資料 §9対応・数値は1問の値）

- **②同訓の語義弁別不全 → L4**。参考: NIER R5（2023）「推し量る」の書き取り1問の正答率は44.5%で、誤答に「押」が目立った。これは**当該1問の値**であり、同訓異字全般・漢字力全般に一般化しない（「生徒は同訓異字ができない」という前提で授業を組まないこと）。L4はこの誤答型（訓は合うが語義が合わない字を書く）を想定し、③語義での見分けを口頭で言わせてから書かせる手順にしている。なお「推し量る」は調査出題語のため、L4〜L7の設問・例文には使用していない（「はかる」系の語自体を本編では出題していない）。 〔根拠区分: Derived〕
- **①文脈×語彙知識の統合不全 → L6**。参考: NIER R7（2025）「かいしん」の変換候補選択1問の正答率は35.7%。これも**当該1問の値**であり、同音異義語識別力全般の値としない。L6は同じ能力（文脈→語義→表記）を書字側から鍛える設計。「かいしん」自体は調査出題語のため設問に使用せず、同型の課題を非調査語（こうえん・しじ・かてい・えいせい・じしん・きかん）で自作した。 〔根拠区分: Derived〕
- **⑥部首・字形の構成要素 → L4主概念2**。部品は「候補想起・意味確認のヒント」と位置づけ、「この部品ならこの意味」という決めつけを本文で明示的に禁止する書き方にした（例外の存在を本文に明記）。 〔根拠区分: Adapted〕
- **③熟語ネットワーク未接続 → L5**。調査出題語「満ち足りた（⇔満足）」は設問に使用しない方針（セルフQA工程の修正記録R-1で明文化済み）を継続し、L5の核字は「決・転・集」で構成した。 〔根拠区分: Rejected〕
- **④過剰な文脈依存 → 全レッスン**。手順④（辞書確認）を全活動の採点条件に組み込み、L6雑談枠・L7問7で辞書使用を成果物として可視化した。 〔根拠区分: Adapted〕
- **強みの足場**: 「落胆する」の意味理解を問う1問の正答率は91.3%（NIER R5・2023。年度対応は内部調査資料 §1.3で確認済み——2026-07-10の外部批判レビューで「年度対応未確認」を訂正）。これも**当該1問の値**であり、「語の意味理解は高い」という領域全体への一般化はしない（内部調査台帳の該当項目・非公開）。L4〜L7も「意味の分かる語」を素材に識別・産出へ進める順序原理としてのみ使う。 〔根拠区分: Derived〕

## 3. 各レッスンの運用メモ

### L04（同訓弁別・部分〜完全書字）

- 導入の「きる」（着る/切る）は口頭のみで扱い、板書で答えを固定しない（両候補が出ること自体が目的）。
- 問2「予定を空ける」・問3「夜が明ける」は、生徒の生活語彙では仮名書きが優勢なことがある。仮名書き解答は「誤り」と言わず、「漢字で書くならどれか」に誘導する。
- 問10（手順の言語化）は本レッスンの評価の本丸。ここが書ければ、個々の書字ミスは回復可能と判断してよい。 〔根拠区分: Experimental〕

### L05（熟語ネットワーク・完全書字）

- 問2「転」で意味の枝が二系統（ころがる／かわる）出るのは想定内で、むしろ気づかせたいポイント。核の字の意味が一つに割り切れない場合があることは、使い分けが割り切れない場合があることと同型の学び。
- 問6は評価規準を「選択の正誤」でなく「場面と言い方の対応＋理由」に置く。模範解答を一つに固定しない（answer_key参照）。 〔根拠区分: Adapted〕
- 中3出口（和語・漢語・外来語）への接続レッスン。中3で本単元を帯再訪する場合はL5→L6を中心に組むとよい。

### L06（語彙選択・完全書字＋産出）

- 活動1は「書く前に③を声に出す」を徹底する。ここを省くと単なる書き取りドリルに退行し、本unitの軸（文脈判断型）から外れる。 〔根拠区分: Adapted〕
- 問8（自分の変換迷い語の持ち寄り）は、R7指導提言の1人1台端末場面（変換候補の漢字の意味の違いを確認する活動）との接続点。interactive HTML（変換候補選択UI・別分冊）を導入する場合、その直後にこの問を置くと接続がよい。
- stretch S1（保証/保障）は意味の重なりが大きい組。断定的な「正解表」を配らないこと。辞書記述に基づく傾向の言語化までで十分。（※この組の公的な使い分け例での位置づけは内部調査資料の範囲で未確認のため、公的資料に帰属させた説明はしない。verify_required）

### L07（統合・産出）

- 校正課題（問1〜6）の誤り例は全て自作。市販教材・調査問題の誤答例の転用はしていない。追加で自作する場合も、単元既習の同訓・同音ペアの範囲で作ると採点が安定する。
- 問6（誤りなし）を必ず残すこと。全問に誤りがある前提で機械的に「直す」構えを崩すのが目的。 〔根拠区分: Experimental〕
- 問8の相互校正は「直すかどうかは書き手が決める」を崩さない。指摘の受け入れを強制すると、使い分けに「言い切れない場合がある」という単元の学習内容と矛盾するメッセージになる。 〔根拠区分: Adapted〕
- 本時は単元の出口評価を兼ねる。評価対象は「字の正誤」より「手順①〜④が運用できているか」「誤りの理由が説明できるか」。

## 4. 帯運用への分割例（10〜15分×複数回）

- L4 → 「活動1のみ」「活動2＋問10」の2回に分割可。
- L5 → 「活動1（核字1つずつ×3回の帯）」＋「活動2」の最大4回に分割可。
- L6 → 「活動1を2組ずつ×3回」＋「活動2」の4回に分割可。朝学習の帯と相性がよい。
- L7 → 分割非推奨（校正→産出→相互点検の流れが一体）。50分確保が難しい場合は活動1を帯で先行し、活動2＋問8を1コマで行う。
- 分割時も4手続き（文脈→候補→語義→辞書）の型は崩さない。辞書（紙または電子）が毎回手元にあることが分割運用の前提条件。

## 5. 事故防止の確認記録（L04〜L07執筆時）

- 調査出題語（推し量る・かいしん・満ち足りた・顧みる・包装）の設問・例文使用: **ゼロ**（執筆時にgrepで確認。2026-07-09時点）。
- 使い分け解説の断定調: answer_keyを含め「一般に〜と書くことが多い」「この文脈では」の限定表現で統一。機械チェック対象語（「必ず」「〜が正解」「〜と書くのが正しい」）の用法断定としての使用ゼロ。L7導入・雑談等の「正解」等の文字列も不使用。
- 常用漢字表への言及: L5雑談枠のみ（「2,136字（平成22年告示）」「目安」）。「基準」「制限」への言い換えなし。制度事実の数値のみ告示年付きで生徒可読部に使用（lesson_map §6の例外規定内）。
- 調査由来の数値: 生徒可読部ゼロ。本ノート§2のみ・調査名＋年度＋「1問の値」の限定付き。
- 雑談枠: 各レッスン1本（L4変換候補が出ない場面／L5・2,136字は掛け算の種／L6辞書の用例欄／L7送信前のひと呼吸）。いずれも内部調査要約指定題材（変換候補・2,136字目安）の範囲内または一般的観察で、未確認の雑学・字源の断定は不使用（字源話自体を使っていない）。
- 字体・読み: 出題語は全て常用漢字表の音訓範囲内であることを執筆時に確認したが、最終的な字体・送りがな表記は教科書照合前提（verify_required）。特に「空ける/明ける」「治す/直す」の送りがな表記は採択教科書の表記と照合すること。
