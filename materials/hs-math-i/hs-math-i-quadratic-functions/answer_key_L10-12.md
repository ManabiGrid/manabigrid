## L10

**問1 (1)** x=−2, 4。
検算: x=−2 で (0)(−6)=0、x=4 で (6)(0)=0。いずれも y=0。

**問1 (2)** x>4。解が表しているもの＝「3x−12>0 を満たすxの値**全部の集まり**」。
検算: x=5 で 3>0 成立、x=4 で 0>0 不成立（境界は含まない）。

**問2** x²−5x＋6>0 → (x−2)(x−3)>0（展開確認: x²−5x＋6 ✓）。下に凸・共有点 x=2, 3。解 **x<2 または 3<x**。
検算: x=0 → 6>0 成立／x=2.5 → 6.25−12.5＋6=−0.25 不成立。

**問3** x²＋x−6≦0 → (x＋3)(x−2)≦0（展開確認: x²＋x−6 ✓）。解 **−3≦x≦2**。
検算: x=0 → −6≦0 成立／x=3 → 6≦0 不成立／端 x=−3 → 0≦0 成立（等号つきなので含む）。

**問4** −x²＋6x−8≧0 → ×(−1)で x²−6x＋8≦0 → (x−2)(x−4)≦0。解 **2≦x≦4**。
検算（元の不等式で）: x=3 → −9＋18−8=1≧0 成立／x=0 → −8≧0 不成立／端 x=2 → −4＋12−8=0≧0 成立。

**問5** x(x−4)>0。共有点 x=0, 4。解 **x<0 または 4<x**。
検算: x=−1 → (−1)(−5)=5>0 成立／x=2 → (2)(−2)=−4 不成立。

**問6**（解答例） 不等式の解とは、不等式を満たすxの値全部の集まりである（前提②）。x²−5x＋6>0 を満たすとは、y=x²−5x＋6 の値が正になること、すなわちグラフ上でその点がx軸より上にあることと同じ。だから解は、グラフがx軸より上にある部分のx座標全部の集まりになる。

**S1** x²−2x−1=0 の解は解の公式より x=(2±√8)/2=1±√2。下に凸・「<0」は間。解 **1−√2<x<1＋√2**。
検算: x=0（1−√2≈−0.41 と 1＋√2≈2.41 の間）→ −1<0 成立／x=3 → 9−6−1=2<0 不成立。x=1±√2 の代入: (1±√2)²−2(1±√2)−1=1±2√2＋2−2∓2√2−1=0 ✓（等号なしなので両端は含まない）。

### L10 本文例題の検算
- 例題1 x²−5x＋4>0 → x<1 または 4<x。x=0→4>0✓／x=2→−2 不成立✓。因数分解 (x−1)(x−4)=x²−5x＋4 ✓。
- 例題2 x²−2x−3≦0 → −1≦x≦3。(x＋1)(x−3)=x²−2x−3 ✓。x=0→−3≦0✓／x=4→5≦0 不成立✓／x=−1→0≦0✓。
- 例題3 −x²＋4x−3>0 → 1<x<3。x=2→1>0✓／x=0→−3>0 不成立✓。

## L11（4パターンはすべて検算）

**問1** y=(x−2)²、下に凸、x=2 で接する。
(1) (x−2)²>0 → **x≠2（x=2 以外のすべての実数）**。検算: x=3→1>0 成立／x=2→0>0 不成立（除く判断と一致）。
(2) (x−2)²≧0 → **すべての実数**。検算: x=2→0≧0 成立／x=0→4≧0 成立。
(3) (x−2)²<0 → **解なし**。検算: x=2→0<0 不成立／x=0→4<0 不成立（どこでも不成立と一致）。
(4) (x−2)²≦0 → **x=2**。検算: x=2→0≦0 成立／x=3→1≦0 不成立。

**問2** x²＋4x＋4=(x＋2)²（展開確認 ✓）≧0 → **すべての実数**。
検算: x=−2→0≧0 成立／x=0→4≧0 成立。

**問3** x²−8x＋16=(x−4)²（展開確認 ✓）<0 → **解なし**。
検算: x=4→0<0 不成立／x=0→16<0 不成立。

**問4** y=x²＋5。頂点 (0, 5) がx軸より上・下に凸 → 共有点0個。
(1) >0 → **すべての実数**（検算: x=0→5>0 成立）。
(2) ≧0 → **すべての実数**（検算: x=0→5≧0 成立）。
(3) <0 → **解なし**（検算: x=0→5<0 不成立。最小値が5なのでどこでも不成立）。
(4) ≦0 → **解なし**（検算: x=0→5≦0 不成立）。

**問5** x²＋2x＋4=(x＋1)²＋3（展開確認: x²＋2x＋1＋3 ✓）。頂点 (−1, 3) がx軸より上・下に凸 → 共有点0個・グラフ全体が上。≦0 → **解なし**。
検算: 最小の値をとる x=−1 でも 3≦0 不成立。

**問6**（解答例） (x−2)² の値は、接点 x=2 でだけ 0 になり、それ以外では正になる。>0 は 0 を含まないので接点を除き、解は x≠2。≧0 は 0 を含むので接点も解に入り、すべての実数になる。違いは接点の1点を含むか除くかだけである。

**S1** y=−(x−1)² は上に凸で x=1 で接し、x=1 以外では y<0、x=1 で y=0。≧0 を満たすのは y=0 の点だけ → **x=1**。
検算: x=1→0≧0 成立／x=0→−1≧0 不成立。

**S2** (x−3)²≦k。k<0 なら左辺の最小値0がkを超え解なし。k>0 なら 3−√k≦x≦3＋√k で1点より広い（例: k=1 なら 2≦x≦4）。**k=0** のとき (x−3)²≦0 → x=3 の1点のみ。答え **k=0**。
検算: k=0, x=3 → 0≦0 成立／x=4 → 1≦0 不成立。

### L11 本文例題の検算
- 4パターン提示 (x−1)²: >0→x≠1（x=1→0>0 不成立、x=0→1>0 成立）／≧0→全実数（x=1→0≧0 成立）／<0→解なし（最小値0で不成立）／≦0→x=1（x=1→0≦0 成立、x=2→1≦0 不成立）。すべて一致 ✓。
- 例題1 x²−6x＋9≦0 → x=3。(x−3)²=x²−6x＋9 ✓。x=3→0≦0✓／x=4→1≦0 不成立✓。
- 例題2 x²−2x＋3>0 → すべての実数。(x−1)²＋2=x²−2x＋3 ✓。最小値2>0 ✓。

## L12

**問1** −5t²＋20t>15 → −5t²＋20t−15>0 → ÷(−5)で向きが変わり t²−4t＋3<0 → (t−1)(t−3)<0（展開確認 ✓）→ 1<t<3。定義域 0≦t≦4 の内側なのでそのまま。解 **1<t<3**。
検算（元の式で）: t=2 → −20＋40=20>15 成立／t=0 → 0>15 不成立／t=1 → −5＋20=15>15 不成立（端を含まない判断と一致）。

**問2** 定義域: 横=8−x>0 かつ x>0 → 0<x<8。条件 x(8−x)≦12 → −x²＋8x−12≦0 → x²−8x＋12≧0 → (x−2)(x−6)≧0（展開確認 ✓）→ x≦2 または 6≦x。定義域と重ねて **0<x≦2 または 6≦x<8**。
検算: x=1 → 面積7≦12 成立／x=4 → 16≦12 不成立／x=2 → 12≦12 成立（端を含む）／x=7 → 7≦12 成立。

**振り返り (1)**（L03） y=x²−4x＋1=(x−2)²−3（展開確認: x²−4x＋4−3 ✓）。頂点 **(2, −3)**。
**振り返り (2)**（L09） 頂点のy座標 −3<0 で下に凸 → x軸をまたぐ → 共有点・実数解は **2個**。
検算: x²−4x＋1=0 の解 x=2±√3 は実数2個 ✓。
**振り返り (3)**（L10） x²−4x＋1>0。共有点 x=2±√3、下に凸、「>0」は外側。解 **x<2−√3 または 2＋√3<x**。
検算: x=0（2−√3≈0.27 より小）→ 1>0 成立／x=2 → −3>0 不成立。

**S1** x(10−x)≧25 → −x²＋10x−25≧0 → x²−10x＋25≦0 → (x−5)²≦0（展開確認 ✓）→ **x=5** の1点のみ（定義域 0<x<10 内 ✓）。L11パターン4（≦0で接点1点）が文脈に現れる例。
検算: x=5 → 面積25≧25 成立／x=4 → 24≧25 不成立。

### L12 本文例題の検算
- 例題1 x(10−x)≧21 → 3≦x≦7。(x−3)(x−7)=x²−10x＋21 ✓。x=3→21≧21 成立／x=5→25≧21 成立／x=2→16≧21 不成立。定義域 0<x<10 に収まる ✓。
