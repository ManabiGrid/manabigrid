## L07 事象への適用

**問1** 縦x cmのとき横は (36−2x)／2 = 18−x cm。S = x(18−x) = −(x−9)²＋81。文脈より x＞0 かつ 18−x＞0、すなわち 0＜x＜18。頂点のx=9は範囲内。**x=9（縦9cm・横9cmの正方形）のとき最大値81cm²**。
検算: x=9 → 9×(18−9)=9×9=81。展開確認: −(x−9)²＋81=−x²＋18x−81＋81=x(18−x)。一致。

**問2** 壁と垂直な辺x mが2本、壁と平行な辺は 12−2x m。S = x(12−2x) = −2x²＋12x = −2(x−3)²＋18。文脈より x＞0 かつ 12−2x＞0、すなわち 0＜x＜6。頂点のx=3は範囲内。**x=3のとき最大値18m²**。
検算: x=3 → 3×(12−6)=3×6=18。展開確認: −2(x−3)²＋18=−2x²＋12x−18＋18=x(12−2x)。一致。

**問3** 条件「壁と平行な辺≦4」より 12−2x≦4、すなわち x≧4。問2の範囲と合わせて **4≦x＜6**。S=−2(x−3)²＋18 は軸 x=3 が区間の左外にあり、x≧4 では減少するから、**x=4 のとき最大値16m²**。
検算: x=4 → 壁と平行な辺は 12−8=4（条件を満たす）、S=4×4=16。式でも −2(4−3)²＋18=−2＋18=16。一致。x=5では 5×2=10＜16 で、減少の向きも確認。

**問4** y = p(100−p) = −(p−50)²＋2500。文脈より価格・売れる本数とも正で 0＜p＜100。頂点のp=50は範囲内。**価格50円のとき最大の売上2500円**。
検算: p=50 → 50×(100−50)=50×50=2500。一致。

**問5** 【説明の要点】式だけの問題なら x はすべての実数を動けるが（この場合 S=−(x−5)²＋25 の最大値は25で同じ値になるものの前提が異なる）、囲いの面積としては縦・横がともに正の長さでなければならないため、範囲は 0＜x＜10 に制限される。**文脈が定義域を決める**という違いを述べていればよい。

**S1（stretch）** 利益 = (p−20)(120−p) = −p²＋140p−2400 = −(p−70)²＋2500。文脈より、売れる杯数が正で p＜120。なお p≦20 でも式自体は定義でき、利益が0以下になるだけである（p＞20 に限定しなくても最大値の結論は変わらない）。頂点のp=70は範囲内。**価格70円のとき最大の利益2500円**。
検算: p=70 → (70−20)×(120−70)=50×50=2500。展開確認: (p−20)(120−p)=120p−p²−2400＋20p=−p²＋140p−2400、−(p−70)²＋2500=−p²＋140p−4900＋2500=−p²＋140p−2400。一致。

## L08 二次方程式とグラフ

**問1** x²−x−6=0 → (x−3)(x＋2)=0 → x=3, −2。共有点は **(3, 0), (−2, 0)**。
検算: x=3 → 9−3−6=0、x=−2 → 4＋2−6=0。両方0。

**問2** −x²＋4x=0 → −x(x−4)=0 → x=0, 4。共有点は **(0, 0), (4, 0)**。
検算: x=0 → 0、x=4 → −16＋16=0。両方0。

**問3** 2x²−8x＋6=0 → 2(x−1)(x−3)=0 → **x=1, 3**。
検算: x=1 → 2−8＋6=0、x=3 → 18−24＋6=0。両方0。

**問4** x²−4x＋1=0 → 解の公式より x=(4±√(16−4))／2=(4±2√3)／2=**2±√3**。
検算: (2＋√3)²−4(2＋√3)＋1=4＋4√3＋3−8−4√3＋1=0。共役側も符号が入れ替わるだけで同様に0。

**問5** **y=(x＋1)(x−5)=x²−4x−5**（展開形でも可）。
検算: x=−1 → 0×(−6)=0、x=5 → 6×0=0。2点 (−1,0), (5,0) を通る。x²の係数1も満たす。

**S1（stretch）** x²−2x−2=2x＋3 → x²−4x−5=0 → (x−5)(x＋1)=0 → x=5, −1。y座標は直線に代入して x=5 → y=13、x=−1 → y=1。共有点は **(5, 13), (−1, 1)**。
検算: 放物線側 x=5 → 25−10−2=13、x=−1 → 1＋2−2=1。直線側と一致。

## L09 x軸との共有点の個数

**問1** y=x²−2x−3=(x−1)²−4。頂点 (1, −4)、下に凸、頂点のy座標が負 → **2個**。方程式で確認: (x＋1)(x−3)=0 → x=−1, 3 の2解。
検算: x=−1 → 1＋2−3=0、x=3 → 9−6−3=0。平方完成の展開確認: (x−1)²−4=x²−2x＋1−4=x²−2x−3。一致。

**問2** y=x²＋6x＋9=(x＋3)²。頂点 (−3, 0) がx軸上 → **1個**。共有点（接点）は **(−3, 0)**。
検算: x=−3 → 9−18＋9=0。展開確認: (x＋3)²=x²＋6x＋9。一致。

**問3** y=−x²＋2x−5=−(x−1)²−4。頂点 (1, −4)、上に凸で頂点がx軸より下 → グラフ全体がx軸より下にあり **0個**。
検算: 展開確認: −(x−1)²−4=−x²＋2x−1−4=−x²＋2x−5。一致。上に凸＋頂点y座標負の組（表の最下段）に該当。

**問4** y=−x²−4x=−(x＋2)²＋4。頂点 (−2, 4)、上に凸で頂点がx軸より上 → **2個**。−x(x＋4)=0 より x座標は **x=0, −4**。
検算: x=0 → 0、x=−4 → −16＋16=0。展開確認: −(x＋2)²＋4=−x²−4x−4＋4=−x²−4x。一致。

**問5** **正しくない**。反例: y=−x²＋2x−5（問3）は頂点 (1, −4) でy座標が負だが、上に凸なので共有点は0個。頂点のy座標の符号だけでは決まらず、**凸の向きとセット**で判定する必要がある、という説明ができていればよい。

**S1（stretch）** y=x²−4x＋k=(x−2)²＋k−4。頂点 (2, k−4)、下に凸。接する ⇔ 頂点のy座標が0 ⇔ k−4=0 ⇔ **k=4**。共有点が2個 ⇔ 頂点のy座標が負 ⇔ k−4＜0 ⇔ **k＜4**。
検算: k=4のとき y=(x−2)² で (2, 0) の1点のみ（L09例題2と同型）。k=0（＜4の例）のとき y=x²−4x=x(x−4) で x=0, 4 の2個。整合。
