## L04 グラフ描画の総合練習

**問1**
(1) y=x²＋2x=(x＋1)²−1。下に凸、軸 x=−1、頂点 (−1, −1)、x=0 のとき y=0。
　検算: x=−1 → 1−2=−1 ✓
(2) y=x²−6x＋5=(x−3)²−4。下に凸、軸 x=3、頂点 (3, −4)、x=0 のとき y=5。
　検算: x=3 → 9−18＋5=−4 ✓
(3) y=x²−1（変形不要）。下に凸、軸 x=0、頂点 (0, −1)、x=0 のとき y=−1（頂点と同じ点）。

**問2** y=2x²−8x＋9=2(x²−4x)＋9=2{(x−2)²−4}＋9=2(x−2)²＋1。
下に凸、軸 x=2、頂点 (2, 1)、x=0 のとき y=9。
検算: 展開 2(x−2)²＋1=2x²−8x＋8＋1=2x²−8x＋9 ✓

**問3** y=−x²＋4x−3=−(x²−4x)−3=−{(x−2)²−4}−3=−(x−2)²＋1。
上に凸、軸 x=2、頂点 (2, 1)、x=0 のとき y=−3。
検算: x=2 → −4＋8−3=1 ✓

**問4** x=2 を y=2x²−8x＋9 に代入: 2·4−16＋9=8−16＋9=**1**。問2の頂点 (2, 1) の y座標と一致 ✓

**問5** 「一般形 → （**頂点**）が読める形に変形 → 凸の向き・軸・（**頂点**）・x=0 のときの y の値を集める → 軸について（**左右対称**）になるように曲線を描く」

**S1** y=−3x²＋6x−1=−3(x²−2x)−1=−3{(x−1)²−1}−1=−3(x−1)²＋2。
上に凸、軸 x=1、頂点 (1, 2)、x=0 のとき y=−1。
y=−3x² のグラフを **x軸方向に＋1、y軸方向に＋2** 平行移動したもの。
検算: 展開 −3(x−1)²＋2=−3x²＋6x−3＋2=−3x²＋6x−1 ✓

## L05 最大値・最小値（定義域の制限なし）

**問1**
(1) y=(x＋1)²−2: 下に凸、頂点 (−1, −2)。**x=−1 で最小値 −2。最大値はない。**
(2) y=x²＋4x＋1=(x＋2)²−3: **x=−2 で最小値 −3。最大値はない。**
　検算: x=−2 → 4−8＋1=−3 ✓
(3) y=−2x²＋8x−5=−2(x²−4x)−5=−2{(x−2)²−4}−5=−2(x−2)²＋3: 上に凸。**x=2 で最大値 3。最小値はない。**
　検算: x=2 → −8＋16−5=3 ✓

**問2** y=x²−1: 下に凸、頂点 (0, −1)。**x=0 で最小値 −1。最大値はない。**

**問3** y=−(x＋1)²: 上に凸、頂点 (−1, 0)。**x=−1 で最大値 0。最小値はない。**

**問4** 誤りは2点。①下に凸の放物線で頂点が与えるのは**最小値**であり、最大値ではない。②定義域に制限がない下に凸の放物線に最大値は存在しない。正しくは「x=3 で最小値 1、最大値はない」。
検算: y=x²−6x＋10=(x−3)²＋1、x=3 → 9−18＋10=1 ✓

**問5** y=x²−2x＋c=(x−1)²＋(c−1)。下に凸なので最小値は c−1。c−1=4 より **c=5**。
検算: y=x²−2x＋5=(x−1)²＋4、最小値 4 ✓

**S1** y=x²＋bx＋7=(x＋b/2)²＋7−b²/4。最小値 7−b²/4=3 より b²=16、**b=±4**。
検算: b=4 → x²＋4x＋7=(x＋2)²＋3、最小値 3 ✓／b=−4 → x²−4x＋7=(x−2)²＋3、最小値 3 ✓
（軸 x=−2 と x=2 が y軸について対称な2通りとして現れる）

## L06 区間つきの最大・最小と場合分け

**問1**
(1) y=x²−4x＋5=(x−2)²＋1（1≦x≦4）。**①軸 x=2 は区間の中**。②候補: x=2 で y=1, x=1 で y=2, x=4 で y=5。③ **x=4 で最大値 5、x=2 で最小値 1**。
　検算: x=1 → 1−4＋5=2 ✓／x=2 → 4−8＋5=1 ✓／x=4 → 16−16＋5=5 ✓
(2) y=x²−6x＋10=(x−3)²＋1（0≦x≦2）。**①軸 x=3 は区間の外**。②候補は両端のみ: x=0 で y=10, x=2 で y=2。③ **x=0 で最大値 10、x=2 で最小値 2**。
　検算: x=2 → 4−12＋10=2 ✓（頂点の値 1 は候補にしない）

**問2** y=−x²＋2x＋3=−(x−1)²＋4（2≦x≦4）。①軸 x=1 は区間の外。②候補: x=2 で y=3, x=4 で y=−5。③ **x=2 で最大値 3、x=4 で最小値 −5**。
検算: x=2 → −4＋4＋3=3 ✓／x=4 → −16＋8＋3=−5 ✓（上に凸・区間内では減少）

**問3** y=−(x−2)²＋4（0≦x≦3）。①軸 x=2 は区間の中。②候補: x=2 で y=4, x=0 で y=0, x=3 で y=3。③上に凸なので **x=2 で最大値 4、x=0 で最小値 0**。
検算: x=0 → −4＋4=0 ✓／x=3 → −1＋4=3 ✓

**問4** ①の判定で軸 x=4 は区間 0≦x≦2 の**外**。よって候補は両端のみで、頂点の値 1 は x=4 が定義域にない以上この関数では実現されない。正しい最小値は 5（x=2 で y=5）。「頂点＝最小」を判定なしに使った点が誤り。

**問5** y=(x−a)²＋2（0≦x≦4）。軸 x=a と区間の位置関係で3つに場合分け。
- (i) a＜0: 軸は左外。**x=0 で最小値 a²＋2**
- (ii) 0≦a≦4: 軸は中。**x=a で最小値 2**
- (iii) a＞4: 軸は右外。**x=4 で最小値 (4−a)²＋2**
検算: x=0 で y=a²＋2 ✓／x=4 で y=(4−a)²＋2 ✓。境界 a=0 では (i)(ii) がともに最小値 2 で連続、a=4 でも (ii)(iii) がともに 2 で一致 ✓

**S1** y=x²−2ax=(x−a)²−a²（0≦x≦2）の最大値。下に凸なので最大の候補は両端: x=0 で y=0, x=2 で y=4−4a。大小の境目は 4−4a=0 すなわち **a=1**（軸が区間の真ん中 x=1 に来るとき）。
- (i) a＜1: **x=2 で最大値 4−4a**（4−4a＞0）
- (ii) a=1: **x=0 と x=2 の両方で最大値 0**（x=0 でも x=2 でも y=0）
- (iii) a＞1: **x=0 で最大値 0**
検算: a=1 のとき x=2 → 4−4=0 で x=0 の値 0 と一致 ✓／a=0 のとき x=2 の値 4 ＞ x=0 の値 0 で (i) と整合 ✓
