# L07 分母に√があるとき

## ねらい

- 分母に√が残った数を、**分母と分子に同じ数をかけて**、分母に√のない形に直せるようになる（この変形を、教科書では**分母を有理化する**と呼ぶ）。
- 変形の目的（近似値の計算がしやすい・形がそろって比べやすい）を実感する。

## 導入：1/√2 は、およそいくつ？

√2÷2 …ではなく、1÷√2、つまり **1/√2** という数を考えよう。これはおよそいくつだろうか。

まともに割ると、1÷1.41421356… という「割る数が無限に続く割り算」になってしまい、手計算ではつらい。ところが、ほんの一手間で、この計算は見違えるほど楽になる。

## 主概念：分母と分子に同じ√をかける

分数は、分母と分子に**同じ数**をかけても値が変わらない（1/2＝2/4 と同じ、小学校以来の性質だ）。そこで 1/√2 の分母と分子に √2 をかける。

1/√2 ＝ (1×√2)/(√2×√2) ＝ √2/2

分母の √2×√2 が (√2)²＝2 になって、√が**分母から消えた**。この変形を、教科書では「分母を**有理化**する」と呼ぶ（分母が有理数になるからだ）。

値が変わっていないことに注意しよう。1/√2 と √2/2 は同じ数の2通りの書き方。でも使い勝手は大違いだ。

√2/2 ≒ 1.414÷2 ＝ 0.707

「1.41421356…で割る」が「2で割る」に化けた。**割る数を√のない数にする**——それがこの変形の実利だ。

もう1つ例を。√3/√5 なら、分母と分子に √5 をかけて、

√3/√5 ＝ (√3×√5)/(√5×√5) ＝ √15/5

:::guide
**かける数は「分母に残っている√」だけでよい**

6/√8 のような形は、いきなり √8 をかけるより、先にL06の変形で √8＝2√2 と中を小さくしてから √2 だけをかける方が数が小さくて済む: 6/(2√2)＝3/√2＝3√2/2。手順として「①まず分母の√の中を簡単にする（L06）②分母に残った√を分母・分子にかける」の順に固定しておくと、大きな数の素因数分解に迷いこまない。なお、この章で必要なのはこの程度までの変形だ——分母に√の和が入るような複雑な形は、ここでは扱わない（必要になったら、そのとき学べば十分）。
:::

:::guide
**「答えはどの形で書くか」問題——形をそろえる意味**

√2/2 と 1/√2 はどちらも正しい値だが、答えの形がばらばらだと、答え合わせのときに同じ数だと気付けない事故が起こる。この教材では「**分母に√を残さない形**」を答えの標準形とする。標準形を決めておく利点は、比較・検算が機械的にできること——たとえば自分の答えが 1/√2 で、解答編の答えが √2/2 だったとき、自分の答えを標準形に直せば「合っていたのに×にしてしまう」事故を防げる。形の約束は、数学の中身というより「通信規格」に近い。
:::

## 練習

1. 次の数の分母を有理化しよう。
   (1) 1/√3　(2) 5/√5　(3) √2/√7
2. 次の数の分母を有理化しよう（先に√の中を簡単にしてから）。
   (1) 3/√12　(2) 10/√50
3. √2≒1.414、√3≒1.732 として、1/√3 の近似値を小数第2位まで求めよう（有理化してから計算すること）。
4. 1/√2 と √2/2 が同じ数であることを、「分数の性質」という言葉を使って一言で説明しよう。

:::zatsudan
1.414…で割るのはうんざりだけど、2で割るのは一瞬——同じ数なのに、書き方ひとつで計算の重さがまるで違う。実は数学の変形って、かなりの部分が「あとで楽をするための引っ越し」なんだ。√の中を小さくするのも（L06）、分母から√を追い出すのも（今日）、全部「計算しやすい家に住み替える」話——数の値は引っ越しても変わらない。変わるのは住み心地！
:::

:::stretch
**S1** 1/√2、1/√3、1/√5 をそれぞれ有理化し、√2≒1.414、√3≒1.732、√5≒2.236 で近似値を出して、3つの数を小さい順に並べよう。並べ終わったら、有理化する前の形（1/√2 など）を見比べて、「分母の√の中が大きいほど、数全体は小さくなる」ことがL03の大小の考え方から説明できるか、考えてみよう。
:::
