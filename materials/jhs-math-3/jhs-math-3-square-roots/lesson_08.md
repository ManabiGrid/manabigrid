# L08 √の加法・減法——予想と反例、そして「1つの数」

## ねらい

- 「√a＋√b＝√(a＋b) ではないか」という自然な予想を立て、**反例で成り立たないことを自分で確かめる**。
- 同じ√どうしを、文字式の同類項と同じ要領でまとめられるようになる。
- √2＋1 のような和も**これ以上簡単にできない「一つの数」** であることを理解し、変形→近似値→大小比較までつなげる。

## 導入：かけ算がうまくいったなら、たし算も？

L05で √2×√3＝√6 が成り立った。中身どうしのかけ算がそのまま通用する、気持ちのいい規則だった。となると、当然こう予想したくなる。

> **予想: どんな数a、bでも √a＋√b＝√(a＋b) が成り立つのではないか？ たとえば √2＋√3＝√(2＋3)＝√5 となるのではないか？**

かけ算で成り立った形が、たし算でも**いつでも**成り立つのでは——とても自然な予想だ。予想を立てたら、次にやることは決まっている。**確かめる**ことだ。

## 主概念1：予想を確かめる——しかし、成り立たない

近似値で実験しよう（L05と同じ、実験→確定の段取りだ）。

- 左辺: √2＋√3 ≒ 1.414＋1.732 ＝ 3.146
- 右辺: √5 ≒ 2.236

3.146 と 2.236。**まるで違う**。この予想は成り立たない。

近似値に頼らない確かめもできる。両辺とも正の数だから、2乗して比べればよい（L03の型）。

- (√2＋√3)² ＝ (√2)²＋2×√2×√3＋(√3)² ＝ 2＋2√6＋3 （前章の乗法公式 (a＋b)²＝a²＋2ab＋b² が√でも働く。これを a＋b√c の形に整理する仕上げは、練習5でキミの手に任せる）
- (√5)² ＝ 5

左辺の2乗には **2√6 というおまけ**が付く。これが0でない限り、両辺は等しくなりようがない。

ここで大事なことをひとつ。「成り立たない」と言い切るには、**成り立たない例（反例）を1つ示せば十分**だった——中2の図形で学んだ、あの反例の考え方だ。いま、√2と√3という反例を1つ確かめたので、「√a＋√b＝√(a＋b) はいつでも成り立つ規則ではない」と堂々と結論してよい。

> **√a＋√b＝√(a＋b) は成り立たない（いつでも成り立つわけではない、ということ。aやbが0のときは等しくなる——stretch S1参照）。√の中身どうしを足してはいけない。**

かけ算・わり算は中身どうしでよいのに、たし算・ひき算はだめ。この非対称は√の世界の重要な地形なので、予想が裏切られた今日の驚きごと覚えておこう。

:::guide
**なぜかけ算はよくて、たし算はだめなのか——正方形の面積で見る**

√を「面積→1辺」の対応と見ると、地形の理由が見えてくる。√a＋√b は「面積aの正方形の1辺」と「面積bの正方形の1辺」をつないだ長さだ。一方 √(a＋b) は「面積a＋bの正方形の1辺」。1辺を単純につないでも、面積が単純に足された正方形にはならない——2つの正方形を合体させて1つの正方形に組み直すとき、辺の長さは足し算では決まらないのだ。かけ算 √a×√b＝√(ab) がうまくいったのは、(√a×√b)²＝ab と**2乗が素直に分配された**から。たし算では (√a＋√b)²＝a＋b＋2√(ab) と、真ん中に交差項 2√(ab) が生まれてしまう。犯人はいつもこの交差項——前章で (a＋b)²≠a²＋b² を学んだときの犯人と同一人物である。
:::

## 主概念2：では、たし算はどうやる？——同類項の要領で

√2＋√3 はこれ以上まとめられない。でも、**同じ√どうし**なら話が別だ。

3√2＋4√2 を考える。√2 というひとかたまりが3個と4個。合わせて7個。

3√2＋4√2 ＝ (3＋4)√2 ＝ 7√2

前章まで散々やった **3x＋4x＝7x と同じ要領**だ。√2 を文字xのように「ひとかたまり」と見て、係数（外の数）だけを足し引きする。ひき算も同様に 5√3−2√3＝3√3。

見た目が違う√も、L06の変形で同じ√が現れることがある。

√50＋√2 ＝ 5√2＋√2 ＝ 6√2

**手順: ①まず各項をa√bの形に直す（L06）②同じ√どうしを集めて係数を足し引きする③違う√どうしは、そのまま並べて答えとする。**

:::guide
**「√2＋√3＝√5」と書きたくなったら、いつでも数直線に戻る**

この誤りは、頭でわかったつもりでも計算の勢いでつい手が出る、しぶとい相手だ。予防線は量感にある。√2は1.4くらい、√3は1.7くらい——足せば3を超える。√5は2.2くらい。**大きさを一言つぶやくだけで、この等式はありえないと体が拒否してくれる**。L06で約束した「変形したら大きさまで言う」は、実はこの場面のための保険でもあった。検算の習慣が誤りを勝手に捕まえてくれる状態が、この章のゴールだ。
:::

## 主概念3：√2＋1 も「一つの数」

では √2＋1 はどうなるだろう。同じ√はないから、もうまとめられない。「計算が終わっていない感じ」がして落ち着かないだろうか？

落ち着かなくてよい理由を言おう。**√2＋1 は、これ以上簡単には表せない、それ自体で完結した一つの数**なのだ。数直線の上では、√2（およそ1.414）から右へ1進んだ、およそ2.414の位置にちゃんと1つの席がある。

これは前章の文字式で a＋b をひとつの式として受け入れたのと同じ感覚だ。a＋b はこれ以上計算できないが、立派に「一つの量」を表していた。√2＋√3 も同じ。**「足しっぱなしに見える形」が、その数の正式な姿**ということがある。

この見方ができると、大小比較の総仕上げができる。たとえば **√3＋√12 と 4 ではどちらが大きい？**

1. 変形する: √12＝2√3 だから √3＋√12＝√3＋2√3＝3√3
2. 2乗して比べる（L03の型）: (3√3)²＝9×3＝27、4²＝16。27＞16 で、どちらも正の数だから **3√3＞4**
3. 感覚でも確認する: √3≒1.732 だから 3√3≒5.196——たしかに4よりはっきり大きい ✓

よって **√3＋√12＞4**。決め手は2の**2乗の比較**だ——27と16という整数の比較になるので、答えが厳密に決まる。3の近似値は、結果が体感と合っているかの**確認役**にまわす（近似値どうしが際どく接近したときも、2乗の比較なら白黒がつく）。変形（L06）→まとめ（今日）→2乗で比較（L03）。この章の道具が全部つながって、一見手が出ない比較が数行で終わる。

:::zatsudan
√2＋1 が「一つの数」だなんて、最初はずるい感じがして「計算しろよ！」と言いたくなるよね。でも考えてみると、πだって「3.14159…と続く、あの数」を1文字で受け入れたのだった（L01）。√2＋1も同じで、「√2より1大きい、あの数」——それで完全に1つの数が指定できている。数の世界では、名前の形はいろいろでも、数直線の席が1つ決まっていれば、それはもう立派な「一つの数」なんだ！
:::

## 練習

1. 次を計算しよう。
   (1) 5√3＋2√3　(2) 7√5−3√5　(3) 4√2＋√3−√2＋2√3
2. 次を計算しよう（まずa√bの形に直してから）。
   (1) √27＋√12　(2) √80−√20　(3) √8＋√50−√32
3. 次の計算はまちがっている。どこがまちがいかを説明し、反例（近似値の計算）でも確かめよう。
   √9＋√16＝√25
4. √5＋√20 と 7 の大小を、本文の3段（変形→2乗して比較→近似値で感覚の確認）で判断しよう（感覚の確認には√5≒2.236を使ってよい）。
5. (√2＋√3)² を前章の乗法公式で展開し、a＋b√c の形で表そう。

:::stretch
**S1** √a＋√b＝√(a＋b) が成り立たないことは確かめた。では、**等号が成り立つ場合は本当に1つもない**のだろうか。a＝0 のときはどうなるか調べてみよう。「a, bがともに正の数のとき」という条件がこの章の議論のあちこちに付いていた理由が、ここから見えてくる。

**S2** 3＋√2 と √2＋3、−1＋√5 と √5−1 のように、同じ数の書き方は何通りもある。「a＋b√c の形に書く」と約束すると答え合わせがしやすくなる。練習5の答えをこの形で書くとき、a, b, cにあたる数を明示してみよう。
:::
