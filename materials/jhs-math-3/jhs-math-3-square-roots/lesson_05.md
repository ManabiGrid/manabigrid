# L05 √の乗法・除法

## ねらい

- √a×√b＝√(ab)、√a÷√b＝√(a/b)（a, b＞0）が成り立つことを、**電卓の実験→2乗による確かめ**の順で納得し、使えるようになる。

## 導入：√どうし、かけ算できるのか？

√2 や √5 を数直線に置き、大小も比べられるようになった。数として一人前に扱うなら、次は**計算**だ。√2×√3 は計算できるのだろうか。できるとしたら、答えはどんな数になる？ まずは予想からいこう。

## 主概念1：実験→予想→確かめ——√a×√b＝√(ab)

**実験。** 電卓で近似値をかけてみる。

√2≒1.414、√3≒1.732 として、1.414×1.732＝2.449048。

一方、√6 を挟み撃ちすると 2.449²＝5.997601、2.450²＝6.0025 だから √6≒2.449。

……そっくりだ。**√2×√3＝√6 ではないか**、と予想が立つ。√の中身どうしのかけ算（2×3＝6）が、そのまま答えの√の中身になっている。

**確かめ。** 近似値の一致は状況証拠にすぎない（L02で「≒と＝は別物」と学んだばかりだ）。ぴったり等しいことは、**2乗**で確かめる。√2×√3 を2乗すると、

(√2×√3)²＝(√2)²×(√3)²＝2×3＝6

つまり √2×√3 は「2乗すると6になる正の数」。それは√6の定義そのものだ。よって

> **a, bが正の数のとき √a×√b＝√(ab)**

これは2と3に限らず、同じ計算でいつでも確かめられる。たとえば √5×√7＝√35、√3×√3＝√9＝3。

:::guide
**「実験→予想→2乗で確定」——この3段は今後もずっと使う**

近似値の実験は発見のためのもの、2乗の計算は確定のためのもの、と役割を分けて覚えておこう。ポイントは、確かめに使った道具が結局 **(√a)²＝a という定義だけ**だったこと。新しい公式に見えるものが、定義から数行で出てくる——だから忘れても自力で復元できる。「公式を忘れたら、2乗してみる」。前章の「忘れたら分配法則に戻る」と同じ系統の安全網だ。
:::

## 主概念2：わり算も同じ形——√a÷√b＝√(a/b)

わり算でも同じことが起こる。√6÷√2 を2乗すると、

(√6÷√2)²＝(√6)²÷(√2)²＝6÷2＝3

「2乗すると3になる正の数」だから √6÷√2＝√3。一般に、

> **a, bが正の数のとき √a÷√b＝√(a/b)　つまり √a/√b＝√(a/b)**

中身どうしを割ればよい。たとえば √35÷√5＝√7、√8÷√2＝√4＝2。

√8÷√2 の例は少し味わい深い。√の付いた数どうしの計算なのに、**答えは2というふつうの整数**になった。√の世界の計算が、ときどき有理数の世界へ「帰ってくる」——中身が平方数になった瞬間だ（L04のguideで見た判定がここでも効いている）。

:::zatsudan
√2×√3の実験、電卓でやってみた？ 1.414×1.732＝2.449048と、√6の近似値2.449…がぴったり重なる瞬間は、何度やってもちょっとぞくっとする。近似値はあくまで「およそ」なのに、その「およそ」どうしのかけ算が、ちゃんと√6の「およそ」に着地する。√たちは、こちらが目を離しているあいだも、きちんと計算のルールを守って暮らしているらしい！
:::

## 気を付けたい形：√の外の数と中の数

3√2 のように、√の外に数が付いた形をこれから多く使う（√2の3倍、という意味だ）。外と中の計算を区別しよう。

- 2√3×4√5＝(2×4)×(√3×√5)＝8√15 （**外どうし・中どうし**をそれぞれかける）
- (2√3)²＝2²×(√3)²＝4×3＝12 （2も√3も両方2乗される）

外の数はふつうの数、中の数は√の世界の数。文字式で 2x×4y＝8xy と係数・文字を分けて計算したのと同じ要領だ。

:::guide
**√(ab)の√は「中身全体」にかかる——書き方の事故を防ぐ**

√2×√3＝√6 を手書きするとき、√の横棒（かさの部分）が6全体を覆うように書こう。ノートで √23 と √2×3 と √(2×3) の区別があいまいになると、見直しのときに自分でも読めなくなる。また 3√2 は「3×√2」の×を省略した書き方であって「√の中に32が入っている」のではない。手書きでは、外の数を√のかさの外に、少し小さめに書く——それだけで事故が激減する。
:::

## 練習

1. 次を計算しよう。
   (1) √3×√7　(2) √5×√5　(3) √12÷√3　(4) √30÷√6
2. 次を計算しよう。
   (1) 3√2×2√7　(2) (3√5)²　(3) 6√10÷3√2
3. √2×√8 を計算すると、答えはどんな数になるだろう。計算してから、「無理数×無理数」の結果について気付いたことを一言書こう。
4. 次の計算はまちがっている。どこがまちがいかを説明し、正しく直そう。
   2√3×3√3＝6√3

:::stretch
**S1** √a×√b＝√(ab) の確かめでは「√a×√b が**正の数**であること」も実は使っている（2乗して6になる数は √6 と −√6 の2つあるのに、√6 の方だと言い切れた理由）。確かめの筋道のどこでこの事実が必要だったか、L01の「√aは正の方」を手がかりに説明してみよう。
:::
