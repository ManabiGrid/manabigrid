# 解答編 L09〜L11（平方根の利用・章末まとめ）

## L09 A判の紙にひそむ√2

### 練習

1. 1:x＝x/2:1 で「内項の積＝外項の積」を使うと、内項の積＝x×(x/2)＝x²/2、外項の積＝1×1＝1。
   よって x²/2＝1、両辺を2倍して **x²＝2**。xは正（長さ）だから x＝√2。
2. 長辺＝20×√2≒20×1.414＝28.28≒**28.3cm**（2乗検算: 1.414²＝1.999396≒2 ✓）
   半分に折ると: 短辺＝28.28÷2≒**14.1cm**（＝10√2）、長辺＝**20cm**（元の短辺がそのまま長辺になる）
   確かめ: 半分の紙の比は 14.14:20＝1:1.414…＝1:√2 で、元と同じ形 ✓
3. 短辺1・長辺2の紙を半分に折ると、短辺2÷2＝1、長辺1 → **1辺1の正方形（比1:1）** になる。
   元の比1:2と半分の比1:1は異なるから、「半分にしても同じ形」は **成り立たない** ✓（1:√2の特別さがわかる）
4. 長辺＝短辺×√2＝√2×√2＝ **2cm**（√が消えて整数に！）
   面積＝√2×2＝ **2√2 cm²**（≒2.83cm²。検算: (√2×2)＝2√2、近似 1.414×2＝2.828 ✓）

### stretch

**S1** 3等分した細長い紙は、短辺＝√2/3、長辺＝1。比を「短:長」でそろえると √2/3:1＝**√2:3**。
元の紙は 1:√2＝√2:2（両方を√2倍）だから、√2:3 とは異なる——**同じ形は保たれない**。
発展: 「3等分しても同じ形」には 1:x で x×(x/3)＝1、x²＝3、**x＝√3**（比1:√3）の紙が必要。「半分→1:√2」「3等分→1:√3」という対応が見える。

## L10 面積から長さへもどる——円の面積の和

### 練習

1. 面積の和＝π×3²＋π×6²＝9π＋36π＝45π。求める半径をxとすると πx²＝45π、x²＝45。
   xは正だから x＝√45＝**3√5 cm**。近似値: 3×2.236＝6.708≒**6.7cm**（2乗検算: 6.708²＝44.997…≒45 ✓）
2. πx²＝18π より x²＝18。x＝√18＝**3√2 cm**（検算: (3√2)²＝9×2＝18 ✓）
3. 1辺＝√12＝**2√3 cm**（(2√3)²＝12 ✓）。面積3cm²の正方形の1辺は√3cm だから、2√3÷√3＝**2倍**。
   （面積が4倍で1辺が2倍、という対応も確認できる: 12＝3×4）
4. まちがい: **半径どうしを足している**。半径10cmの円の面積は100πで、25π＋25π＝50πと等しくない。
   正しくは πx²＝50π、x²＝50、x＝√50＝**5√2 cm**（≒7.07cm。検算: (5√2)²＝50 ✓）。
   長さの足し算を面積の世界に持ちこんではいけない——√の中身が足せないのと同じ地形。

### stretch

**S1** ドーナツ形の面積＝36π−16π＝20π。πx²＝20π より x²＝20、x＝√20＝**2√5 cm**（≒4.47cm。検算: (2√5)²＝20 ✓）
**S2** √20＝√(4×5)＝√4×√5＝2√5。一般に √(4a)＝√4×√a＝**2√a** だから、面積が4倍になると1辺はちょうど2倍になる。「面積のかけ算は、√を通すと外に出るときに√4＝2という倍率に化ける」——L05の√a×√b＝√(ab)の応用そのもの。

## L11 章末まとめ——数の拡張の物語・第2幕をふり返る

### 総合演習

1. 負の側: −√5≒−2.236＜−2。正の側: (√3)²＝3＜4＝2²＜6＝(√6)² より √3＜2＜√6。
   小さい順: **−√5, −2, 0, √3, 2, √6**
2. (1) √45−√20＋√5＝3√5−2√5＋√5＝**2√5**（検算: 近似 6.708−4.472＋2.236＝4.472＝2×2.236 ✓）
   (2) √6×√8÷√3＝√(6×8÷3)＝√16＝**4**
   (3) (√5−√2)(√5＋√2)＝(√5)²−(√2)²＝5−2＝**3**（前章の和と差の積の公式が√でも働く）
3. 2/√6＝(2×√6)/(√6×√6)＝2√6/6＝**√6/3**。近似値: 2.449÷3＝0.8163…≒**0.82**（検算: 2.449²＝5.997601≒6 ✓）
4. ①変形: √28＝2√7 だから √7＋√28＝√7＋2√7＝3√7
   ②2乗して比較: (3√7)²＝9×7＝**63**、8²＝**64**。63＜64 で、どちらも正の数だから 3√7＜8。よって **√7＋√28＜8**
   ③感覚の確認（近似値）: 3√7≒3×2.646＝7.938＜8 ✓（2乗検算: 7.938²＝63.01…≒63＝(3√7)² ✓）——63対64のかなり際どい勝負。こういうときこそ、近似値の目分量ではなく**2乗の比較**が白黒を厳密につけてくれる
5. 1辺＝√48＝√(4²×3)＝**4√3 cm**。整数の挟み込みは直接 36＜48＜49（6²＜48＜7²）から **6＜√48＜7**、つまり6と7の間（7にかなり近い）。（近似確認: 4×1.732＝6.928 ✓）
6. (1) **×**。反例: √9＋√16＝3＋4＝7 だが √(9＋16)＝√25＝5。7≠5。
   (2) **×**。反例: √2×√8＝√16＝4 は有理数（無理数どうしの積が有理数になることがある）。
   (3) **○**。2乗して0になる数は0だけ（正の数の平方根が2つあるのとは違う、0だけの特別ルール）。

### stretch

**S1**（アイデアの入り口）この証明の方針は「**偶数・奇数に注目する**」こと。√2＝m/n（これ以上約分できない分数）と仮に置いて両辺を2乗すると 2n²＝m² となり、m² が偶数——つまり m も偶数だとわかる（奇数の2乗は奇数だから）。ここから先、m と n の偶数・奇数を追いかけていくと、「これ以上約分できない」という最初の設定とぶつかる衝突が起こる——その続きは「ルート2 無理数 証明」で調べて、自分のノートで再現してみよう（証明の全体をきちんと組み立てる作法は、高校の「背理法」で学ぶ）。
**S2** x²＝18 を満たすxは「18の平方根」だから **x＝√18 と −√18**、変形して **x＝3√2 と −3√2**（±3√2）。検算: (±3√2)²＝9×2＝18 ✓。正の方だけ答えるのは長さの場面（L10）の作法であって、方程式では**2つとも**答える——次章で最初に確認されるポイント。
