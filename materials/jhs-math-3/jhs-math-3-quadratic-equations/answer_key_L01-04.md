# 解答編 L01〜L04（二次方程式とその解き方 前半）

## L01 方程式の物語・第3幕——二次方程式とその解

### 準備運動（前提診断）

1. 3x＝15 より **x＝5**（検算: 3×5−7＝8 ✓）
2. 2式をたすと 2x＝8 → x＝4、y＝7−4＝3。**x＝4、y＝3**（検算: 4−3＝1 ✓）
3. (1) **(x＋3)(x＋4)**（展開して x²＋7x＋12 に戻る ✓）　(2) **(x＋3)(x−3)**（和と差の積）
4. (1) **√7 と −√7**（±√7。「平方根」と聞かれたら2つ）　(2) √16＝**4**（正の方1つ。4²＝16 ✓）
5. **x²＋3x−10**

### 練習

1. **ア・エ**。
   アは ax²＋bx＋c＝0 の形そのもの。エは展開・整理すると x²＋2x−3＝0 で二次方程式。
   イは一次方程式。**ウは x² が消える罠**——整理すると 0＝4x−1、つまり一次方程式（見た目に x² があっても、整理して残らなければ二次方程式ではない）。
2. 順に代入する。
   x＝−3: 9＋3−6＝6 ≠0（解でない）／x＝−2: 4＋2−6＝**0 ✓ 解**／x＝2: 4−2−6＝−4 ≠0（解でない）／x＝3: 9−3−6＝**0 ✓ 解**
   **解は −2 と 3**（候補4つのうち2つ——「一般に解は二つ」の実感）。
3. x＝−3 を各式に代入。
   (ア) (−3)²−9＝0 ✓／(イ) (−3)²＋3×(−3)＝9−9＝0 ✓／(ウ) (−3)²−(−3)−6＝9＋3−6＝6 ≠0
   **アとイ**。（1つの値が複数の方程式の解になることもある。）
4. 左辺: 6²＝36。右辺: 4×6＋12＝36。**左辺＝右辺が成り立つので、x＝6 は解である** ✓

### stretch

**S1** 代入すると、x＝0: 0²＝0 ✓／x＝1: 1²＝1 ✓／x＝−1: 1≠−1／x＝2: 4≠2。**解は 0 と 1**。
答案のまずい点: **xが0である可能性があるのに、両辺をxで割った**こと。0で割る計算は存在しないから、「xで割る」はx≠0のときにしか許されない操作であり、割った瞬間に x＝0 という解が失われた。正しくは移項して x²−x＝0 → x(x−1)＝0（この解き方はL05で学ぶ）。

## L02 平方根の考えで解く——ax²＝b と (x＋m)²＝n

### 練習

1. (1) x²＝49 → **x＝±7**（(±7)²＝49 ✓）
   (2) x²＝6 → **x＝±√6**（(±√6)²＝6 ✓）
   (3) 両辺を5で割って x²＝9 → **x＝±3**（5×9＝45 ✓）
   (4) 両辺を4で割って x²＝25/4 → **x＝±5/2**（4×25/4＝25 ✓）
2. (1) x＋3＝±5 → x＝2 または x＝−8。**x＝2、x＝−8**（代入: (2＋3)²＝25 ✓、(−8＋3)²＝(−5)²＝25 ✓）
   (2) x−1＝±√7 → **x＝1±√7**（代入: (1±√7−1)²＝(±√7)²＝7 ✓）
   (3) x＋4＝±√12＝±2√3 → **x＝−4±2√3**（(±2√3)²＝12 ✓）
   (4) x−5＝±√8＝±2√2 → **x＝5±2√2**（(±2√2)²＝8 ✓）
3. 1辺を x cm（x＞0）とすると x²＝50。x＝±√50＝±5√2。長さだから正の方をとって **5√2 cm**。（検算: (5√2)²＝25×2＝50 ✓。大きさ: 7²＝49＜50＜50.41＝7.1² なので、7と7.1の間——およそ7.1cm。）
4. まちがい: **x＝−4 を落としている**。16の平方根は 4 と −4 の2つ。正しくは **x＝±4**。（面積の問題なら「長さだから4」と選ぶ理由を書いた上で−4を除くが、方程式 x²＝16 を解くだけなら両方が解。）

### stretch

**S1** 2x−1＝±3。
2x−1＝3 → 2x＝4 → x＝2。2x−1＝−3 → 2x＝−2 → x＝−1。**x＝2、x＝−1**
（代入: (2×2−1)²＝3²＝9 ✓、(2×(−1)−1)²＝(−3)²＝9 ✓）

## L03 平方の形に変形して解く——x²＋px＋q＝0

### 練習

1. xの係数10の半分は5、その2乗は**25**。
   x²＋10x＋**25**＝1＋**25** → (x＋**5**)²＝**26**（展開の逆確認: (x＋5)²＝x²＋10x＋25 ✓）
2. (1) x²＋2x＝5 → 両辺に1²＝1を加えて (x＋1)²＝6 → **x＝−1±√6**
   （代入検算: x＝−1＋√6 のとき x²＋2x−5＝(7−2√6)＋(−2＋2√6)−5＝0 ✓。もう一方も同様）
   (2) x²−6x＝−4 → 両辺に(−3)²＝9を加えて (x−3)²＝5 → **x＝3±√5**
   （x＝3＋√5: x²−6x＋4＝(14＋6√5)−(18＋6√5)＋4＝0 ✓）
   (3) x²＋8x＝−9 → 両辺に16を加えて (x＋4)²＝7 → **x＝−4±√7**
   （x＝−4＋√7: x²＋8x＋9＝(23−8√7)＋(−32＋8√7)＋9＝0 ✓）
   (4) x²−4x＝1 → 両辺に4を加えて (x−2)²＝5 → **x＝2±√5**
   （x＝2＋√5: x²−4x−1＝(9＋4√5)−(8＋4√5)−1＝0 ✓）
3. x²＋12x＝−2 → 半分6の2乗36を両辺に加えて (x＋6)²＝34 → **x＝−6±√34**
   （x＝−6＋√34: x²＋12x＋2＝(70−12√34)＋(−72＋12√34)＋2＝0 ✓）

### stretch

**S1** x²＋5x＝−2。xの係数5の半分は5/2、その2乗は25/4。両辺に加えて
(x＋5/2)²＝−2＋25/4＝17/4
x＋5/2＝±√17/2 → **x＝(−5±√17)/2**
（検算は次レッスンの解の公式でも一致: a＝1、b＝5、c＝2 で √(25−8)＝√17、x＝(−5±√17)/2 ✓）

## L04 解の公式——時短の道具とその出どころ

### 練習

1. (1) a＝1、b＝5、c＝3。√の中: 25−12＝13。**x＝(−5±√13)/2**
   （代入: x＝(−5＋√13)/2 のとき、平方の形の逆確認 (x＋5/2)²＝13/4 → x²＋5x＋25/4＝13/4 → x²＋5x＋3＝0 ✓）
   (2) a＝1、b＝−3、c＝−2。−b＝−(−3)＝3、√の中: 9＋8＝17。**x＝(3±√17)/2**
   （逆確認: (x−3/2)²＝17/4 → x²−3x＋9/4＝17/4 → x²−3x−2＝0 ✓）
   (3) a＝2、b＝3、c＝−1。√の中: 9＋8＝17。分母は2a＝4。**x＝(−3±√17)/4**
   （代入: 2x²＋3x＝2×(26∓6√17)/16＋(−9±3√17)/4＝(13∓3√17)/4＋(−9±3√17)/4＝4/4＝1、よって 2x²＋3x−1＝0 ✓）
   (4) a＝3、b＝−4、c＝−2。√の中: 16＋24＝40。x＝(4±√40)/6＝(4±2√10)/6＝ **(2±√10)/3**
   （√40＝2√10 に整理してから、分子のすべての項と分母を2で約分）
2. a＝1、b＝−6、c＝3。√の中: 36−12＝24。x＝(6±√24)/2＝(6±2√6)/2＝**3±√6**
   （逆確認: (x−3)²＝6 → x²−6x＋3＝0 ✓）
3. a＝1、b＝1、c＝−6。√の中: 1＋24＝25＝5²。x＝(−1±5)/2 → **x＝2、x＝−3**
   因数分解では x²＋x−6＝(x＋3)(x−2)＝0 → x＝−3、x＝2。**同じ解になる** ✓（代入: 4＋2−6＝0 ✓、9−3−6＝0 ✓）
4. まちがい: **b＝−3 なのに b＝3 として代入した**（xの係数は符号ごと読む）。正しくは a＝1、b＝−3、c＝1 で、−b＝3、√の中: 9−4＝5。**x＝(3±√5)/2**
   （誤答 (−3±√5)/2 を代入してみると: x＝(−3＋√5)/2 のとき x²＝(9−6√5＋5)/4＝(7−3√5)/2、−3x＝(9−3√5)/2 なので、x²−3x＋1＝(7−3√5)/2＋(9−3√5)/2＋1＝(16−6√5)/2＋1＝9−3√5。√5 は2.2…だから 9−3√5 は約2.3で、0にならない——誤答は方程式を満たさない。さらに逆確認をすると (x＋3/2)²＝5/4 → x²＋3x＋1＝0 となり、誤答は**符号の違う別の方程式の解**だと分かる。）

### stretch

**S1** x²＋px＋q＝0 → x²＋px＝−q → 両辺に (p/2)² を加えて
(x＋p/2)²＝p²/4−q＝(p²−4q)/4
x＋p/2＝±√(p²−4q)/2 → **x＝(−p±√(p²−4q))/2**
（一般の公式で a＝1、b＝p、c＝q とおいた結果と一致する ✓）
