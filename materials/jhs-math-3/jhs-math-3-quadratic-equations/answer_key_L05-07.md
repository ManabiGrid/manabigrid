# 解答編 L05〜L07（因数分解・解法選択・立式）

## L05 因数分解で解く——「または」の論理

### 練習

1. (1) x＋1＝0 または x−6＝0 → **x＝−1、x＝6**（代入: (0)(−7)＝0 ✓、(7)(0)＝0 ✓）
   (2) x＝0 または x＋8＝0 → **x＝0、x＝−8**（代入: 0×8＝0 ✓、(−8)×0＝0 ✓）
2. (1) (x＋2)(x＋5)＝0 → **x＝−2、x＝−5**（代入: 4−14＋10＝0 ✓、25−35＋10＝0 ✓）
   (2) (x−4)(x＋3)＝0 → **x＝4、x＝−3**（代入: 16−4−12＝0 ✓、9＋3−12＝0 ✓）
   (3) (x−5)²＝0 → **x＝5 の1つだけ**（代入: 25−50＋25＝0 ✓）
   (4) (x＋7)(x−7)＝0 → **x＝±7**（代入: 49−49＝0 ✓。x²＝49 として平方根の考えでも同じ解）
   (5) x(x＋4)＝0 → **x＝0、x＝−4**（代入: 0＋0＝0 ✓、16−16＝0 ✓）
   (6) まず＝0に整理: x²−2x−8＝0 → (x−4)(x＋2)＝0 → **x＝4、x＝−2**（もとの式で検算: 16−8＝8 ✓、4＋4＝8 ✓）
3. まちがい: **右辺が0でないのに「積＝3だから各因数＝…」とした**こと。「AB＝0ならばA＝0またはB＝0」は右辺が0のときだけ使える（積が3になる2数の組は無数にある）。
   正しくは展開・整理する: x²−4x＋3＝3 → x²−4x＝0 → x(x−4)＝0 → **x＝0、x＝4**
   （検算: x＝0: (−3)(−1)＝3 ✓／x＝4: (1)(3)＝3 ✓。誤答の確認: x＝6 では (3)(5)＝15≠3、x＝2 では (−1)(1)＝−1≠3——代入検算がまちがいを見抜いてくれる。）

### stretch

**S1** (x−3)(x＋7)＝0 を展開して **x²＋4x−21＝0**（これが答えの一例。両辺を何倍かした 2x²＋8x−42＝0 なども正解）。
確かめ: x＝3: 9＋12−21＝0 ✓／x＝−7: 49−28−21＝0 ✓

## L06 解き方を選ぶ——三つの道具の使い分け

### 練習（選ぶ理由は一例。別の選択でも、理由が筋なら正解）

1. x²＝18 → **平方根の考え**（x²＝数 の形そのまま）。x＝±√18＝**±3√2**（(±3√2)²＝18 ✓）
2. x²＋9x＋14＝0 → **因数分解**（和が9・積が14の組: 2と7 がすぐ見える）。(x＋2)(x＋7)＝0 → **x＝−2、x＝−7**（代入: 4−18＋14＝0 ✓、49−63＋14＝0 ✓）
3. x²＋3x−3＝0 → **解の公式**（積が−3・和が3の整数の組がない）。a＝1、b＝3、c＝−3。√の中: 9＋12＝21。**x＝(−3±√21)/2**（逆確認: (x＋3/2)²＝21/4 → x²＋3x−3＝0 ✓）
4. (x−7)²＝5 → **平方根の考え・かたまりのまま**（展開したら損）。x−7＝±√5 → **x＝7±√5**（(±√5)²＝5 ✓）

### stretch

**S1** どちらの道でも **x＝−1±√5**。
平方の形: x²＋2x＝4 → (x＋1)²＝5 → x＝−1±√5（3行）
解の公式: √の中: 4＋16＝20、x＝(−2±√20)/2＝(−2±2√5)/2＝−1±√5（√の整理と約分が1段多い）
（代入検算: x＝−1＋√5 のとき x²＋2x−4＝(6−2√5)＋(−2＋2√5)−4＝0 ✓）
好みはどちらでもよい——「係数が偶数だから平方の形が速い」「公式なら迷わない」など、理由が言えていれば正解。

## L07 方程式を「つくる」——二通りに表して等号で結ぶ

### 練習A（つくるだけ。着目した数量の言い方は一例）

1. **x(x＋1)＝72**（着目: 2数の積。①式で表すと x(x＋1) ②問題文から 72）
2. **x²＝5x＋24**（着目: その数を2乗した結果。①x² ②5倍より24大きい数として 5x＋24）
3. **x(x＋4)＝60**（着目: 長方形の面積。①縦×横 ②60）
4. **(x＋3)²＝100**（着目: 新しい正方形の面積。①1辺(x＋3)の2乗 ②100）
5. **x(14−x)＝45**（着目: 2数の積。和が14だから、一方がxなら他方は 14−x）

### 練習B（一例）

x(x−5)＝24 に合う場面の例:
「横が x m、縦が横より5m短い長方形の面積が24m²」／「ある正の数xと、それより5小さい数の積が24」など。
（xと、xより5小さい数の積が24——という骨組みが読めていれば、肉付けは自由。ただしxが長さや正の数になる場面では、解の吟味のときに x＞5 が効くことになる。）

### 練習C

1. x(x＋1)＝72 → x²＋x−72＝0 → 和が1・積が−72の組: 9と−8 → (x＋9)(x−8)＝0 → x＝8、x＝−9。
   xは正の整数だから x＝8。**答え 8と9**（検算: 8×9＝72 ✓）
4. (x＋3)²＝100 → x＋3＝±10 → x＝7、x＝−13。
   xは長さだから x＝7。**答え 7cm**（検算: (7＋3)²＝100 ✓）

### stretch

**S1** (ア) x(14−x)＝45 → 14x−x²＝45 → 整理して x²−14x＋45＝0 → (x−5)(x−9)＝0 → x＝5 または x＝9。
x＝5 なら他方は 14−5＝9、x＝9 なら他方は 14−9＝5。**どちらの解からも、2つの数は5と9**（検算: 5＋9＝14 ✓、5×9＝45 ✓）。
(イ) もう一方をxとおいても、残りは 14−x なので、式は **x(14−x)＝45 と同じ形**になる。この問題は2つの数の立場が対等（和と積しか条件がない）なので、どちらをxとおいても式ごと同じになる——おき方で式が変わる問題（L09 stretch）と比べてみると発見がある。
