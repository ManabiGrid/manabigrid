# L13 章末まとめ——中学の関数、完成

## ねらい

- この章の道具一式を、**入口**（事象から関数を見いだす）と**出口**（特徴を説明する）を中心に総合演習で点検する。
- 中1から続いた関数の学びを見わたし、次章「相似」への橋を確認する。

## この章の地図

学んだことを、一枚に圧縮しておこう。

| 道具 | 中身 | レッスン |
|---|---|---|
| 定義と見分け方 | y＝ax²。見分け方は2本立て（m倍→m²倍・y÷x²が一定）。1組の値（x≠0の組。(0, 0)はどのy＝ax²も通るので決められない）から式が決まる | L01〜L02 |
| 数学化の型 | 変わる量を選ぶ→表→きまり→式。変域は場面が決める | L03・L10 |
| グラフ | 対応の集まり。放物線。aの符号=向き・絶対値=開き具合 | L04〜L05 |
| 値の変化 | 増減は原点を境に変わる。変域の検算3点セット。変化の割合は一定でない——だから曲線 | L06〜L08 |
| 説明の型 | 観点＋比較＋根拠。「求めたら、読む」 | L08〜L09 |
| モデル化 | 理想化して「みなす」→予測→確かめ→食い違いの原因を探る | L11 |
| 関数の完成 | 式に表すことが困難な関数も、ただ1つに決まる対応なら関数 | L12 |

太い柱は2本。**事象から関数を見いだす力**（入口）と、**特徴を言葉で説明する力**（出口）だ。式の計算やグラフの作図は、この2本の柱をつなぐ廊下にあたる。章末問題も、この2本を重点的に確かめる構成にしてある。

## 3年間の物語を閉じる

中1で、関数という考え方に出会った。xを決めるとyがただ1つ決まる、あの定義だ。比例と反比例という最初の2人の登場人物。中2で、比例を仲間にふくむ一次関数へ広がり、変化の割合という「ものさし」を手に入れた。そして中3。変化の割合が一定でない関数y＝ax²に出会い、さらに式に表すことが困難な関数までを関数の仲間として迎え入れた。

ふり返ると、物語を貫いていた問いはずっと同じだ。**「一方を決めると、もう一方はどう決まるか。」** 式の形は幕ごとに変わっても、この問いだけは変わらなかった。中学の関数の学びはここでひと区切り——だが、量と量の関係を追いかけるこの問い自体は、この先の数学でも、理科でも、ずっと現役で働き続ける。

## 次章への橋——面積は「2乗に比例」で読める

最後に、すぐ隣にある続きを1つ予告しておこう。円の面積の公式S＝πr²は、この章の目で見ると「**Sはrの2乗に比例する関数**（比例定数π）」と読める。だからm倍→m²倍の性質がそのまま使えて、半径が2倍になれば面積は4倍、3倍なら9倍——計算しなくても言い切れる。次の章「相似」では、形を拡大・縮小したときの面積の比がテーマの1つになる。そこでこの「2乗の目」が、思いがけない場所からもう一度登場する。楽しみにしていてほしい。

:::zatsudan
道具箱がいっぱいになった日のことを想像する。金づち1本だった頃は、何を見てもくぎに見えた。道具が増えると、世界の見え方の方が変わってくる——料金表を見れば階段のグラフが浮かび、車の速度計を見れば2乗の伸びを思う。数学を学ぶとは、結局のところ「見えるものが増えること」なのかもしれない。この章で君の目に増えたものが、これから思わぬ場所で働くのを楽しみにしていよう。
:::

## 章末問題

1. yはxの2乗に比例し、x＝−2のときy＝8である。
   (1) yをxの式で表そう。 (2) x＝3のときのyの値を求めよう。
2. 【入口】底面が1辺x cmの正方形で、高さが5cmの直方体がある。この直方体の体積をy cm³とする。
   (1) 数学化の型にそって、yをxの式で表そう（表はx＝1, 2, 3で作り、y÷x²の商も調べること）。
   (2) yはxの2乗に比例するといえるか。比例定数も答えよう。
   (3) xの変域を答えよう。
3. y＝−2x²について答えよう。
   (1) グラフは上と下のどちらに開くか。
   (2) x＞0の範囲では、xの値が増えるとyの値はどうなるか。
   (3) xが1から2まで増えるときの変化の割合を求めよう。
4. y＝x²/2について、xの変域が−2≦x≦4のときのyの変域を求めよう（検算の3点セットを使うこと）。
5. 【グラフ】y＝x²、y＝2x²、y＝−x²の3つのグラフを同じ座標平面にかき、それぞれに式のラベルを付けよう（各グラフとも、x＝−2, −1, 0, 1, 2に対応する点をとってから結ぶこと）。
6. 【出口・説明】y＝3x²の変化の割合は一定でない。xが0から1まで増えるときと、1から2まで増えるときの変化の割合をそれぞれ計算し、その結果をグラフの形と結びつけて、説明の型（観点＋比較＋根拠）で3文以内で説明しよう。
7. 【いろいろな関数】架空のコインロッカーは、預ける時間が「6時間まで」なら300円、「6時間をこえて12時間まで」なら500円、「12時間をこえて24時間まで」なら800円である。預ける時間をx時間（0＜x≦24）、料金をy円とするとき、yはxの関数といえるか。中1の関数の定義を使って理由を説明しよう。

:::stretch
**S1** 円の面積S＝πr²を「Sはrの2乗に比例する関数」とみて、次の問いに**計算せずに**答えよう。
(1) 半径が2倍になると、面積は何倍になるか。
(2) 半径が3倍になると、面積は何倍になるか。
(3) 半径10cmの円の面積は、半径5cmの円の面積の何倍か。
答えたら、πr²の計算で1つだけ検算してみよう。「公式を関数として読む」このワザが、次章の相似で面積比を考えるときの主役になる。
:::
