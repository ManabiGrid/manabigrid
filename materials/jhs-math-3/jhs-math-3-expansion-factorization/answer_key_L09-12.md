## L10 式による説明①

### 練習

1. nを整数とすると、連続する2つの奇数は 2n＋1, 2n＋3、間の偶数は 2n＋2 と表される。
   (2n＋1)(2n＋3)＋1＝4n²＋6n＋2n＋3＋1＝4n²＋8n＋4＝4(n²＋2n＋1)＝4(n＋1)²＝(2(n＋1))²＝ **(2n＋2)²**
   よって、連続する2つの奇数の積に1をたすと、間の偶数の2乗になる。
   検算（例）: 3×5＋1＝16＝4² ✓、7×9＋1＝64＝8² ✓
2. nを整数とし、連続する3つの整数を n−1, n, n＋1 と表す。
   (n−1)＋n＋(n＋1)＝**3n**
   3n は真ん中の数 n の3倍だから、連続する3つの整数の和は真ん中の数の3倍になる。
   検算（例）: 5＋6＋7＝18＝3×6 ✓
3. ゴールの形: 「2つの整数の和」＝ n＋(n＋1)。nを整数とし、続いた2つの整数を n, n＋1 と表す。
   (n＋1)²−n²＝n²＋2n＋1−n²＝2n＋1＝n＋(n＋1)
   よって、2つの続いた整数の2乗の差は、その2つの整数の和に等しい。
   検算（例）: 6²−5²＝11＝5＋6 ✓
   （別解: 和と差の積で (n＋1)²−n²＝(n＋1＋n)(n＋1−n)＝(2n＋1)×1——因数分解でゴールの形が直接現れる）
4. 不備: 偶数を 2n、奇数を 2n＋1 と**同じ文字**で表すと、「6と7」のようにとなり合う偶数・奇数の組しか表せない。「4と11」のような離れた組を含む**すべての場合**を扱うには、別々の文字を使う。
   直し: mとnを整数とし、偶数を 2m、奇数を 2n＋1 と表す。和は 2m＋(2n＋1)＝2(m＋n)＋1。m＋n は整数だから、2(m＋n)＋1 は奇数。よって偶数と奇数の和は奇数である。

### stretch

**S1** n(n＋8)−(n＋1)(n＋7)＝(n²＋8n)−(n²＋7n＋n＋7)＝(n²＋8n)−(n²＋8n＋7)＝**−7**
結果に n が残らないので、田の字を並びのどこで取っても、対角線どうしの積の差は常に −7（左上×右下のほうが7小さい）で変わらない。
検算（例）: n＝1 → 1×9−2×8＝9−16＝−7 ✓

## L11 式による説明②

### 練習

1. もとの正方形の面積は x²。長方形の面積は (x＋2)(x−2)＝x²−4。
   x²−4 は x² より4小さい。よって長方形の面積はもとの正方形より **4cm²小さくなる**。
2. のばす・縮める長さを a とすると、長方形の面積は (x＋a)(x−a)＝x²−a²——もとの正方形より a² 小さい。
   式を読む: 差 a² には **x が入っていない**。つまり「同じ量だけのばして縮めると、面積は必ず小さくなり、減る量は（正方形の大きさによらず）**のばした長さの2乗**になる」。
3. ステップ①: 道の外側は縦 p＋2a・横 q＋2a の長方形。
   S＝(p＋2a)(q＋2a)−pq＝pq＋2ap＋2aq＋4a²−pq＝2ap＋2aq＋4a²＝**2a(p＋q＋2a)**
   ℓ＝真ん中の一周＝縦 (p＋a)・横 (q＋a) の長方形の周＝2{(p＋a)＋(q＋a)}＝**2(p＋q＋2a)**
   よって aℓ＝a×2(p＋q＋2a)＝2a(p＋q＋2a)＝S。**S＝aℓ は長方形でも成り立つ** ✓
4. S＝π(r＋a)²−πr²＝π(r²＋2ar＋a²)−πr²＝π(2ar＋a²)＝**πa(2r＋a)**
   ℓ＝2π(r＋a/2)＝**π(2r＋a)**
   aℓ＝a×π(2r＋a)＝πa(2r＋a)＝S。**円でも S＝aℓ が成り立つ** ✓

### stretch

**S1** (x＋a)(x−b)＝x²＋(a−b)x−ab。これが x² に等しいためには、
(a−b)x−ab＝0　すなわち　**(a−b)x＝ab**
読み取れること（例）:
- a＝b（同じ量ののばし縮め）だと左辺が0になり、ab＞0 と矛盾——練習2のとおり、同じ量では絶対に同じ面積にならない。
- 同じ面積になるには a＞b（のばすほうを長く）が必要で、そのとき x＝ab/(a−b)。**x が式に残っている**＝この条件は正方形の大きさに関係する。たとえば a＝2, b＝1 なら x＝2 の正方形のときだけ面積が等しくなる（確かめ: (2＋2)(2−1)＝4＝2² ✓）。
