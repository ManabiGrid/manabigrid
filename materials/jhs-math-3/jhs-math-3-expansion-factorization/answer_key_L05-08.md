# 解答編 L05〜L08（因数分解）

## L05 積の形のよさと因数分解——共通因数

### 練習

1. (1) 4x＋8y＝**4(x＋2y)**　検算: 4(x＋2y)＝4x＋8y ✓
   (2) x²−7x＝**x(x−7)**　検算: x(x−7)＝x²−7x ✓
   (3) 6a²＋9ab＝**3a(2a＋3b)**　検算: 3a(2a＋3b)＝6a²＋9ab ✓
   （3(2a²＋3ab) や a(6a＋9b) は共通因数が残っていて不完全）
   (4) 5x²−10x＋15＝**5(x²−2x＋3)**　検算: 5(x²−2x＋3)＝5x²−10x＋15 ✓
   （かっこの中 x²−2x＋3 は、共通因数がなく、積3・和−2になる整数の組もないので、ここで完了）
2. (1) 3a²＋6ab＋3a＝**3a(a＋2b＋1)**　最後の項 3a÷3a＝1 を忘れない。
   検算: 3a(a＋2b＋1)＝3a²＋6ab＋3a ✓（項数3 ✓）
   (2) 4xy−2y＝**2y(2x−1)**　検算: 2y(2x−1)＝4xy−2y ✓
3. 不完全な点: かっこの中 4x²＋6x に、まだ共通因数 **2x** が残っている。
   最後まで: 8x²＋12x＝**4x(2x＋3)**　検算: 4x(2x＋3)＝8x²＋12x ✓
4. ① ab＋3a＋2b＋6＝a(b＋3)＋2(b＋3)
   ② 共通のかっこ (b＋3) をくくり出して ＝ **(b＋3)(a＋2)**
   検算: (b＋3)(a＋2)＝ab＋2b＋3a＋6 ✓

### stretch

**S1** 前2項と後2項で組にする: xy−x＋y−1＝x(y−1)＋(y−1)＝ **(y−1)(x＋1)**
組の作り方を変える（1・3項目と2・4項目）: xy＋y−x−1＝y(x＋1)−(x＋1)＝(x＋1)(y−1)——同じ答え ✓
検算: (y−1)(x＋1)＝xy＋y−x−1＝xy−x＋y−1 ✓

## L06 公式による因数分解①

### 練習

1. (1) 積20の組: (1,20)(2,10)(4,5)。和9は (4,5) → **(x＋4)(x＋5)**　検算: x²＋9x＋20 ✓
   (2) 積16の組: (1,16)(2,8)(4,4)。和10は (2,8) → **(x＋2)(x＋8)**　検算: x²＋10x＋16 ✓
   (3) 積＋15・和−8 → 両方マイナス: (−3,−5) → **(x−3)(x−5)**　検算: x²−8x＋15 ✓
   (4) 積＋36・和−12 → (−6,−6)。2数が同じなので **(x−6)²**　検算: x²−12x＋36 ✓
2. (1) 積−15 → 異符号。組: (5,−3)で和2 ✓ → **(x＋5)(x−3)**　検算: x²＋2x−15 ✓
   (2) 積−18・和−3 → (−6,3) → **(x−6)(x＋3)**　検算: x²−3x−18 ✓
   (3) 積−30・和−1 → (−6,5) → **(x−6)(x＋5)**　検算: x²−x−30 ✓
3. 積が18になる正の整数の組と、その和:
   (1,18)→19、(2,9)→11、(3,6)→9。よって **□＝19, 11, 9**
4. 検算すると (x＋7)(x−2)＝x²＋5x−14 となり、xの項の符号が合わない（＋5x になってしまう）。
   正しくは、積−14・和−5 → (−7,2) → **x²−5x−14＝(x−7)(x＋2)**　検算: x²＋2x−7x−14＝x²−5x−14 ✓

### stretch

**S1** 積36の正の組と和: (1,36)→37、(2,18)→20、(3,12)→15、(4,9)→13、(6,6)→12。
よって **p＝37, 20, 15, 13, 12 の5個**。
36を30に変えると: (1,30)→31、(2,15)→17、(3,10)→13、(5,6)→11 の4個。16なら: (1,16)→17、(2,8)→10、(4,4)→8 の3個。
観察: pの候補の個数は「積の数を2つの正の整数の積に分ける分け方の個数」＝**約数のペアの個数**と一致する（36のように平方数だと (6,6) のような同数ペアが1つ入る）。

## L07 公式による因数分解②

### 練習

1. (1) 両端: x²と64＝8²。真ん中: 2×x×8＝16x ✓ → **(x＋8)²**　検算: x²＋16x＋64 ✓
   (2) 両端: x²と25＝5²。真ん中: 2×x×5＝10x ✓ 符号− → **(x−5)²**　検算: x²−10x＋25 ✓
   (3) 両端: a²と81＝9²。真ん中: 2×a×9＝18a ✓ → **(a＋9)²**　検算: a²＋18a＋81 ✓
2. (1) x²−36＝x²−6²＝ **(x＋6)(x−6)**　検算: x²−36 ✓
   (2) x²−121＝x²−11²＝ **(x＋11)(x−11)**　検算: x²−121 ✓
   (3) a²−49＝a²−7²＝ **(a＋7)(a−7)**　検算: (a＋7)(a−7)＝a²−7a＋7a−49＝a²−49 ✓
   (4) x²−y²＝ **(x＋y)(x−y)**　検算: (x＋y)(x−y)＝x²−xy＋xy−y²＝x²−y² ✓
3. **(ウ) x²＋6x＋16** が平方の形にならない。
   理由: 両端は x² と 16＝4² だが、真ん中の点検で 2×x×4＝8x≠6x。さらに2数探しに回しても、積16・和6になる整数の組は存在しない（(1,16)→17、(2,8)→10、(4,4)→8）ので、これ以上因数分解できない。
   （参考: (ア)は 2×x×6＝12x ✓ で (x＋6)²、(イ)は 2×x×4＝8x ✓ で (x＋4)²）
4. 100＝10² だから a＝10。真ん中の項は ±2×x×10＝±20x。よって **□＝20 または −20**（(x＋10)² または (x−10)²）。

### stretch

**S1** (n＋1)²−n²＝(n＋1＋n)(n＋1−n)＝(2n＋1)×1＝**2n＋1**
読み取り: 連続する2つの整数の平方の差は、**その2つの整数の和**（＝必ず奇数）になる。
確かめ: 7²−6²＝49−36＝13＝6＋7 ✓　50²−49²＝2500−2401＝99＝49＋50 ✓

**S2** (1) 9a²−16b²＝(3a)²−(4b)²＝ **(3a＋4b)(3a−4b)**
検算: (3a＋4b)(3a−4b)＝9a²−12ab＋12ab−16b²＝9a²−16b² ✓
(2) 25a²−1＝(5a)²−1²＝ **(5a＋1)(5a−1)**
検算: (5a＋1)(5a−1)＝25a²−5a＋5a−1＝25a²−1 ✓

## L08 いろいろな因数分解

### 練習

1. (1) 共通因数2 → 2(x²−9)。かっこの中は平方−平方 → **2(x＋3)(x−3)**
   検算: 2(x²−9)＝2x²−18 ✓　完了チェック: 各因数はこれ以上分解できない ✓
   (2) 共通因数3 → 3(a²＋6a＋9)。3点点検（2×a×3＝6a ✓）→ **3(a＋3)²**
   検算: 3(a²＋6a＋9)＝3a²＋18a＋27 ✓
   (3) 共通因数4x → **4x(x＋2)**　検算: 4x²＋8x ✓
   (4) 共通因数a → a(x²−x−12)。積−12・和−1 → (−4,3) → **a(x−4)(x＋3)**
   検算: a(x²−x−12)＝ax²−ax−12a ✓
2. (1) a＋b＝M と置く。M²−16＝(M＋4)(M−4)＝ **(a＋b＋4)(a＋b−4)**
   検算: (M＋4)(M−4)＝M²−16 ✓
   (2) x−2＝M と置く。M²＋8M＋16＝(M＋4)²＝(x−2＋4)²＝ **(x＋2)²**
   検算: (x＋2)²＝x²＋4x＋4。もとの式を展開して確認: (x−2)²＋8(x−2)＋16＝x²−4x＋4＋8x−16＋16＝x²＋4x＋4 ✓ 一致
3. (1) 展開: (x−4)(x＋9)＝x²＋5x−36。因数分解し直す: 積−36・和5 → (9,−4) → (x＋9)(x−4) ✓ もとに戻った
   (2) 展開: 5(x＋2)²＝5(x²＋4x＋4)＝5x²＋20x＋20。因数分解し直す: 共通因数5 → 5(x²＋4x＋4)、3点点検 → 5(x＋2)² ✓ もとに戻った
4. (1) **完了している**。検算: (x−6)(x＋1)＝x²−5x−6 ✓。各因数（x−6、x＋1）は1次式でこれ以上分解できず、共通因数もない。
   (2) **完了していない**。かっこの中 x²−4 が平方−平方。続き: 2(x²−4)＝**2(x＋2)(x−2)**　検算: 2(x²−4)＝2x²−8 ✓

### stretch

**S1** は設問ではなく、レッスン内でM置き換えの1段目（x⁴−16＝(x²＋4)(x²−4)）までを示した「発展の入り口」のため、解答はなし（この先の分解は高校範囲として本文で予告のみ）。

**S2** 説明の例: L字形の面積は x²−9（大きい正方形から1辺3の正方形を除いた残り）。L字形を点線で切って2つの長方形に分け、片方を回して並べ替えると、縦 (x−3)・横 (x＋3) の長方形ができる。切って並べ替えても面積は変わらないから、x²−9＝(x−3)(x＋3)。
つまりこの図は、公式 x²−a²＝(x＋a)(x−a) の a＝3 の場合を「面積の保存」で見せている——式の変形（因数分解）が、図形の「切って並べ替え」に対応している。
