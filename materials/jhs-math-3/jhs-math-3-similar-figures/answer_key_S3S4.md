# 解答集（自己採点用）— 3節・4節（L12〜L16）

> 自己採点用（生徒も見てよい。解いてから開こう）。

## L12 面積比は相似比の2乗

**例題1**: (1) 4:9 (2) 45cm²（20×9/4。検算: 45/20=2.25=(3/2)²）
**例題2**: 相似比2:3→面積比4:9→△ABC=36×4/9=**16cm²**

**練習**
1. 面積比 **9:25**。大きい方 18×25/9=**50cm²**（検算: 50/18=25/9）
2. **1:16**
3. 面積比16:25=4²:5² → 相似比**4:5**。対応辺 8×5/4=**10cm**

**stretch**: 縦だけ3倍=面積3倍、相似（縦横3倍）=9倍。差の正体は「横の3倍」の掛け残し。個数（頂点・対角線の本数など）は相似で不変——長さでも面積でもない量は比の対象にならない。

## L13 立体の相似・表面積の比

**例題**: 1:9 → 40×9=**360cm²**

**練習**
1. **4:25**
2. 3組の辺が1:2, 2:4, 3:6ですべて1:2→相似・相似比**1:2**、表面積比**1:4**。実測: A=2(1·2+2·3+1·3)=**22cm²**、B=2(2·4+4·6+2·6)=**88cm²**。88/22=4 ✓
3. 9:16=3²:4² → 相似比**3:4**。対応辺 6×4/3=**8cm**

**stretch**: 円柱小=2π·2²+2π·2·3=20π cm²、大=2π·4²+2π·4·6=80π cm²。80π/20π=4 ✓。球はどの2つも相似（中心から一定倍の拡大で重なる。相似比=半径の比）。

## L14 体積比は相似比の3乗

**例題1**: 8:27 → 40×27/8=**135cm³**（検算: 135/40=3.375=(3/2)³）
**例題2**: **大サイズが割安**（量3.375倍・価格3倍。体積1あたり小50円・大44.4…円）

**練習**
1. **1:64**
2. 体積比27:125 → 54×125/27=**250cm³**（検算: 250/54=125/27）
3. 相似比1:2→体積比**1:8**。価格は900/150=**6倍**。量8倍に対し価格6倍→**大が割安**（体積1あたり小150/1=150円相当・大900/8=112.5円相当）
4. 表面積比**4:9**、体積比**8:27**

**stretch**: 体積1あたり同額なら大は400×27/8=**1350円**。材質・厚さが同じで材料の量が表面積に比例するという仮定のもとで——表面積（材料の目安）は2乗・体積（中身）は3乗で増える→大型化するほど中身/材料の比が相似比倍で有利になる。

## L15 縮図による測量

**例題（本文）検算メモ**: 木=10·tan35°≈7.00m→縮図3.50cm（1/200）→+1.5m=**約8.5m** ✓／池=余弦定理でAB=√(18²+24²−2·18·24·cos60°)=√468≈21.63m→縮図7.21cm（1/300）→**約21.6m** ✓

**練習**
1. 30m→1/500で6cm。25°で作図、縦≈**2.8cm**（真値: 30·tan25°≈13.99m→2.80cm）→実長約14m→+1.5m=**約15.5m**
2. AB=20m→1/400で5cm。∠A=90°・∠B=54°で作図、AP≈**6.9cm**（真値: 20·tan54°≈27.53m→6.88cm）→**約27.5m**
3. 解答例: かいた縮図は実際の図形と**相似**である。相似な図形では**対応する線分の比**がすべて等しく、その比が**縮尺**にあたる。だから縮図上で測った長さを縮尺で割り戻せば（縮尺の逆数倍すれば）、実際の長さが求められる。

**stretch**: 縮尺1/250なら縮図上の長さは1/500の2倍（12cm・縦約5.6cm）だが、実長に戻せば同じ約14m＋1.5m——縮尺を変えても答えは変わらない（読み取り精度は上がる→L16へ接続）。影を使う代替測定: 同時刻の「木の影の長さ」と「長さの分かる棒とその影」の組（太陽光が平行→相似な直角三角形ができる）。

## L16 誤差と活用の深まり

**例題（本文）**: 0.1×200=20cm（0.2m）のずれ。

**練習**
1. 0.1×500=**50cm**
2. **1/100の方が小さい**。同じ0.1cmの読み違いが、1/100では実長10cm、1/500では50cmのずれになる（ずれは縮尺の逆数倍で拡大されるため、縮図が大きい=縮尺の分母が小さいほど影響が小さい）。
3. 解答例（2つで可）: ①縮図と実物が相似であることを相似の性質（対応する線分の比）で根拠づけられる ②縮図をかかずに比例式・計算で長さを求められる ③答えに含まれる誤差を見積もり、近似値として適切に表せる。
**stretch**: 水平距離に比例する（角固定）のは**目の高さから木の先端までの部分**（約7.00m）で、10→10.1mならこの部分が1%増えて約7.07m。目の高さ1.5mは変わらないので、木の高さの答え全体では約8.502m→約8.572m、増加率はおよそ**0.8%**（≒0.07÷8.50。「答え全体が1%増える」と書かないこと——比例するのは目の高さから上の部分だけ）。工夫の例——測定: 巻尺をたるませない・複数回測って平均／作図: 大きめの縮尺・鋭い鉛筆／読み取り: 目盛りを正面から読む・複数人で読む。
