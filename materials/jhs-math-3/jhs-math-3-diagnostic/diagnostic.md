# 中3数学・はじめる場所を見つける地図

## この診断について——最初に読んでみよう

これは**テストではない**。点数をつけて優劣を決めるものでも、合否を判定するものでもない。これから中3数学を独学で進めるあなたが、**「自分はどこから始めるのがいちばん得か」を自分で見つけるための地図**だ。

数学は積み上げの教科だから、いま立っている場所がわからないまま歩き出すと、遠回りになる。逆に、止まっている段がわかれば、そこだけ埋めて先へ進める。だから——間違えた問題こそ、この診断でいちばん価値のある収穫だと思ってほしい。

なお、本診断は**試行版**だ。問題の難しさや判定の基準（閾値）は、まだ実際の利用データによる較正（こうせい: ちょうどよい値への調整）を行っていない。実利用のデータがたまりしだい較正し、改訂していく予定である。

**使い方**

1. Q1から**順番に**解いてみよう。この順番は、中1から中3へ学ぶ系統の順に並べてある。途中を飛ばしたり、順番を変えたりしないこと（順番そのものが診断の仕掛けだから）
2. 解答用の紙かノートに、式と答えを書く。頭の中だけで済ませず、途中の式も書いておくと、あとで「どこでずれたか」が自分で見える
3. わからない問題は、5分考えてもだめなら「わからない」と書いて次へ進もう。それも大事な記録だ（この「5分」は考え込んだときの打ち切りの上限で、ふつうの1問の目安時間1〜1.5分とは別のものだ）
4. 全部終わったら（途中の段までで区切ってもよい）、解答編（**diagnostic_answers**）で自己採点する。採点のしかたと、結果から始める場所を決める**判定表**もそこにある。画面で使う場合は、問題と解答を隣に並べると行き来しやすい。なお、途中の段までで区切った場合、そこまでの結果による判定は**暫定判定**になる。最後に解いた段が停止候補（段の過半数を誤り）だったときは、判定を確定するために、その次の段まで解いてから区切ろう
5. 目安は解答が約22〜33分（1問あたり1〜1.5分）＋採点10分ほど。1回で終わらなくてよい。段の切れ目で中断して、続きは後日でもかまわない

:::guide
**なぜ「やさしい順」ではなく「学ぶ順」なのか**

この診断の問題は、やさしい順に並んでいるのではなく、**学校で学ぶ系統の順**に並んでいる。数学のつまずきは多くの場合「ある段から先が急にわからなくなる」形で起こる。学ぶ順に解いていくと、正解が続いたあと、ある地点から誤りが続き始める——その**切り替わりの地点**が、あなたの学び直しの開始地点だ。ばらばらの順で出題すると、この切り替わりが見えなくなってしまう。
:::

## 段10 中3への入口

最後の3問は、中3の新しい内容ではない。これまでに学んだ道具が、**中3の入口3単元（式の展開と因数分解・平方根・関数y=ax²）の入口の形**で使えるかを確かめる問題だ。

**Q20** 次の式を計算してみよう。

−3x(2x−5)

**Q21** 次の2つに答えてみよう。

(1) 2乗すると16になる数を**すべて**書こう。
(2) 面積が10cm²の正方形がある。この正方形の1辺の長さは、整数（cm）で表せるだろうか。「表せる／表せない」で答え、そう考えた理由も短く書いてみよう。

**Q22** y＝x² という式について、次の表を完成させよう。

| x | −2 | −1 | 0 | 1 | 2 |
|---|----|----|---|---|---|
| y |    |    |   |   |   |

:::guide
**段10の3問が見ているもの**

Q20は分配法則。中3最初の単元「式の展開と因数分解」は、この計算を多項式どうしに広げるところから始まる。Q21は「2乗の逆」を考える感覚。「平方根」の単元は、まさに(2)のような問いに名前をつける物語だ。答えを見る前に、2乗の意味に立ち返って自分の考えを書いておこう。Q22は2乗の計算と対応表で、「関数y=ax²」の単元の一歩目そのものである。この3問がすらすら解けたなら、中3の入口はもう目の前にある。
:::
