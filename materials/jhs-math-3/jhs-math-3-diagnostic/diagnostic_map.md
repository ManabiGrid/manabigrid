# 診断マップ — 中3数学「はじめる場所を見つける地図」設計文書

## 0. このモジュールは何か

設計判断の記録に基づく初実装。**単元ごとの入口診断は作らず、科目（中3数学）ごとに1本**の診断を置く。目的は選抜でも成績づけでもなく、独習者が「どこまで理解できていて、どの段階から学び直せばよいか」を**自分で**つかむこと。本診断は**試行版**（項目難易度・停止閾値は較正前の初期値）であり、実利用データがたまりしだい較正・改訂する。

- 出題範囲は**前学年（中1・中2）の内容を含む**——中1相当から始める（診断設計様式1）
- 出題順は**学習する系統順に固定**（診断設計様式2）。シャッフルしない。系統順に解き進めることで「どの段階で理解が止まっているか」が段階的に見える
- 判定表は「ここまで正解→この単元・このレッスンから始める」の形で、止まった段階と開始地点を直結させる（診断設計様式3）

## 1. 系統の段構成（10段・22問）

中学数学の系統は完全な一本線ではなく、**「数と式」の幹**に **「関数」「図形」「データ」の枝**が寄り添う形をしている。本診断は、幹（段1・2・3・5・6）に枝の起点（段4の関数、段8の図形、段9のデータ）を系統順で挟み込む固定順で出題する。同じ枝の中では学年順（中1→中2）。

| 段 | 領域・学年 | 問 | 出題意図 |
|---|---|---|---|
| 段1 | 中1 正負の数 | Q1〜Q2 | すべての計算の土台。四則と累乗（Q1）、数直線・絶対値の**意味**（Q2）。ここが崩れていると以降すべてが連鎖的に崩れる |
| 段2 | 中1 文字と式 | Q3〜Q4 | 文字式の計算（Q3）と、**場面を文字式に翻訳する力**（Q4）。「計算はできるが立式できない」型のつまずきを分離して見る |
| 段3 | 中1 一次方程式 | Q5〜Q6 | 解く技能（Q5）と、「解とは代入して成り立つ値」という**解の意味**（Q6）。意味の欠落は中3二次方程式の「解の吟味」で表面化する |
| 段4 | 中1 比例・反比例 | Q7〜Q8 | 関数の枝の起点。式の決定（Q7）と、対応・変化の**関係の読み**（Q8）。一次関数→y=ax²へ続く枝の根元 |
| 段5 | 中2 式の計算 | Q9〜Q10 | 多項式の加減・単項式の乗除（Q9）と、**文字式で数の性質を説明する**入口（Q10）。中3「式の展開と因数分解」の直接の前提 |
| 段6 | 中2 連立方程式 | Q11〜Q12 | 解く技能（Q11）と場面からの立式（Q12）。中3二次方程式の「方程式をつくる」（二次方程式L07）に直結する前提 |
| 段7 | 中2 一次関数 | Q13〜Q14 | 変化の割合（Q13）と2点からの式決定（Q14）。y=ax²で「変化の割合が一定**でない**」ことに驚くための足場 |
| 段8 | 中1・中2 図形 | Q15〜Q17 | 多角形の内角（Q15・中1）、三角形の内角と外角（Q16・中2）、**合同条件の識別**（Q17・中2）。相似・円周角・三平方の枝の根元。Q17の選択肢には「2組の角がそれぞれ等しい」（合同条件ではなく相似条件）を混ぜ、相似L03への伏線とする。なお本段が測るのは角度計算と合同条件の識別のみで、証明構成・図の読解は測定範囲外（§6参照） |
| 段9 | 中1・中2 データ・確率 | Q18〜Q19 | 相対度数（Q18・中1）と確率の計算・**確率の意味**（Q19・中2）。標本調査の枝の根元。「確率1/4＝4回に必ず1回」という誤解の検出を含む |
| 段10 | 中3の入口3単元 | Q20〜Q22 | **中3の新出内容は問わない**。既習の道具が「中3の入口の形」で使えるかを見る。Q20＝分配法則の運用（→式の展開の入口）、Q21＝2乗の逆と「整数で表せない長さ」（→平方根の入口）、Q22＝2乗の対応表（→関数y=ax²の入口） |

設問数: 段1〜7・9が各2問、段8・10が各3問、計**22問**。各段とも「計算・技能」1問＋「意味理解」1問を基本の対にする（段8・10は3問構成）。

## 2. 判定の論理

**初めて連続で誤り始めた地点＝学び直しの開始地点**、を基本原理とする。運用規則は次の3つ。

1. **段単位で見る**: 段の全問正解＝通過。段の1問だけ誤り＝「要注意」（先へ進んでよいが、その分野を軽く見直す）。段の過半数を誤り＝その段で**停止**の候補
2. **連続性で確定する**: 停止候補の段の**次の段が通過でなければ**（過半数誤りに加え、1問誤りの「要注意」の場合も含む＝安全側・より手前から始める側に倒す暫定規則）、最初の停止候補の段が学び直しの開始地点。次の段が通過なら、停止候補は「その分野だけの穴」として個別復習扱い（幹は生きている）。最終段（段10）には次の段がないため、段10の過半数誤りはそのまま停止として確定する。最初の開始地点が確定したあとも、後方に停止候補の段が現れた場合は、開始地点は変えずに「追加の穴」として記録し、判定表の該当行の対応を併記する。複数の枝で停止した場合は、最も早い段の停止を主経路とし、行き先単元のL01の直接の前提にあたる段（例: y=ax²に対する段4・7）は「あわせて復習」として併記する。なお「次の段」による確定は、系統順の隣接段で機械的に見る簡便法であり、隣の段が異なる領域（枝）の場合も形式上は確定に使われる限界がある（安全側に倒れる設計で吸収し、領域の前提関係（DAG）に基づく判定への再設計は判定表v2で扱う）
3. **枝は独立に判定する**: 図形（段8）やデータ（段9）の停止は、数と式の幹（段1・2・3・5・6。段4は関数の枝）の判定に影響しない。幹が通っていれば中3は始められる——枝の学び直しは、対応する中3単元に入る**前まで**にやればよい

この論理により、「全部やり直し」という過剰な差し戻しを避け、**止まった枝だけ**を戻る設計になる。

## 3. 判定表の行き先（実在レッスンとの対応）

本パッケージの中3数学8単元と、その学習順（教科書の標準的な配列に合わせた系統順）:

| 順 | 単元 | 最初のレッスン（実在） |
|---|---|---|
| ① | 式の展開と因数分解 | L01 単項式と多項式の乗法・除法（**冒頭に前提診断枠あり**） |
| ② | 平方根 | L01 新しい数が必要になるとき——平方根と√（**冒頭に準備運動＝前提診断あり**） |
| ③ | 二次方程式 | L01 方程式の物語・第3幕——二次方程式とその解 |
| ④ | 関数y=ax² | L01 関数の物語・第3幕——2乗に比例との出会い |
| ⑤ | 相似な図形 | L01 相似の意味と記号∽ |
| ⑥ | 円周角 | L01 円のふしぎ——動かしても変わらない角 |
| ⑦ | 三平方の定理 | L01 定理を見いだす——3つの正方形 |
| ⑧ | 標本調査 | L01 全部は調べられない——全数調査と標本調査 |

判定表（詳細はdiagnostic_answers.md）の対応方針:

- **止まった段がなかった場合（要注意の段があってもよい）** → ①式の展開と因数分解L01から（すでに中3を学習中の人は、上の表から自分の位置の単元へ）
- **段1〜3（中1・数と式の幹）で停止** → 中1内容の学び直しが先。**中1・中2教材は本パッケージに未整備**のため、「調べるフレーズ」で橋渡しし、学び直し後に①L01へ
- **段5で停止** → 中2式の計算の学び直し→①L01へ（L01の前提診断枠で再確認できる）
- **段6で停止** → ①②から始めてよいが、③二次方程式L01の**前まで**に連立を学び直す
- **段4・7（関数の枝）で停止** → ①から始めてよいが、④関数y=ax² L01の**前まで**に比例・一次関数を学び直す
- **段8（図形の枝）で停止** → ①から始めてよいが、⑤相似L01の**前まで**に図形（角度・合同・証明）を学び直す
- **段9（データの枝）で停止** → ①から始めてよいが、⑧標本調査L01の**前まで**にデータ・確率を学び直す
- **段10の個別誤り** → Q20は①L01、Q21は②L01がそのまま受け皿（前提診断枠は①〜④・⑧のL01に実在する）。Q22は①L01から始め、④関数y=ax² L01の**前まで**に関数の枝（比例→一次関数）を復習する（④L01冒頭の前提診断枠でも再確認できる）
- **段10で2問以上誤り（過半数）** → ①式の展開と因数分解L01から順に始める（中3の入口から丁寧に。判定表に専用行を置く）

**正直さの規律**: 中1・中2の学び直し先教材は本パッケージに存在しない。判定表では「戻り先教材は今後整備予定。当面は検索・AIチャット・手持ちの教科書で」と明記し、検索フレーズ例・AI宛プロンプト例を併記する（独習原則v2.3③・丸投げにしない）。AI回答は教科書・本診断の解答と照合してから使う注意も判定表側に明記する（検証済みの学び直し資料リストの整備は将来課題）。

## 4. 所要時間の目安

- 解答: 22問 × 約1〜1.5分 ＝ 約22〜33分
- 自己採点＋判定表の読み取り: 10分前後
- **合計 約32〜43分（30〜45分を見込んでおけば足りる）**（1回で終わらなくてよい。段の切れ目で中断・再開できる構成にする）
- 診断本文の「1問5分」は考え込んだときの**打切り上限**であり、上の標準時間（1問1〜1.5分）とは別の目安である

## 5. 設問設計の規律

1. **全設問自作**（教科書・過去問・調査問題の転載ゼロ）。数値・場面はすべて本モジュールのために新規に作成
2. **自己採点可能な形式のみ**: 計算・短答・選択・表の穴埋め。説明を求める設問（Q10等）は「模範解答と見比べて、要点の語が入っているか」で自己採点できる形にする
3. **各段に意味理解の設問を混ぜる**: 計算だけの診断は「手続きは覚えているが意味が抜けている」型のつまずきを見逃すため
4. **媒体等価**（設計原則）: 画面・紙どちらでも同じに使える。図が必須の設問は置かない（図形の段も、言葉と数値だけで完結する設問にする）
5. **調査データの数値は引用しない**: 本モジュールはパスA調査を経ていないため、難易度・誤答傾向の数値根拠は本文にもteacher_notes相当にも書かない（確実でない引用をしない規律）。設問の狙いは設計判断として記述する
6. **文体はv2.4準拠**: 「〜してみよう」基本形・——は最小限・学年超え漢字の初出にふりがな括弧書き・【ことば】枠は最小限
