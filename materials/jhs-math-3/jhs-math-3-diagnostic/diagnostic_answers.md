# 中3数学診断 解答・自己採点・判定表

問題本体は **diagnostic.md**。画面で使う場合は、問題と本ファイルを隣に並べると行き来しやすい。

## 自己採点のしかた

1. 各問を「正解／誤り／わからないと書いた」に分ける。**「わからない」は誤りと同じ扱い**にする（できていない場所を知ることが目的だから、正直に数えるほど診断は正確になる）
2. 小問が複数ある問題（Q1など）は、**全小問正解でその問1問の正解**とする。1つでも落としたらその問は誤り。きびしく感じるかもしれないが、診断はきびしめに数えるほうが安全側だ（本当はできるのに復習する損より、できないのに先へ進む損のほうが大きい）
3. 説明を書く問題（Q10(2)・Q21(2)）は、模範解答の **〔採点のポイント〕に挙げた要点が入っているか**で判定する。一字一句同じである必要はない
4. 段ごとに「2問中何問正解か」（段8・10は3問中何問か）をメモしてから、最後の判定表へ進もう

## 解答と検算

### 段1 正負の数

**Q1**
(1) −7＋12＝**5**
(2) 3−(−5)＝3＋5＝**8**
(3) (−4)×(−6)＝**24**（負×負は正）
(4) (−2)³＝(−2)×(−2)×(−2)＝4×(−2)＝**−8**

検算: (1) 5＋7＝12 ✓　(4) 指数の3は「−2を3回かける」の意味。(−2)²＝4、4×(−2)＝−8 ✓

**Q2**
(1) **2**のほうが大きい（数直線で右にある数が大きい。負の数はすべて正の数より小さい）
(2) **4と−4**の2つ（絶対値は数直線上で0からの距離。距離が4になる点は0の右と左に1つずつ）

〔採点のポイント〕(2)は「4」だけでは誤り。**2つとも**書けて正解。

### 段2 文字と式

**Q3**
(1) 3x＋5−x＋2＝(3x−x)＋(5＋2)＝**2x＋7**
(2) 2(a−3)−(a＋1)＝2a−6−a−1＝**a−7**

検算: (2)にa＝10を代入。もとの式: 2×7−11＝14−11＝3。答えの式: 10−7＝3 ✓

**Q4** 出した金額から代金を引く。代金は80×x＝80x（円）だから、おつりは
**1000−80x（円）**

検算: x＝5なら代金400円、おつり600円。式: 1000−80×5＝600 ✓
〔採点のポイント〕「80x−1000」（引く向きが逆）は誤り。単位の（円）はなくても正解でよい。

### 段3 一次方程式

**Q5**
(1) 4x−7＝5 → 4x＝12 → **x＝3**
(2) 3(x−2)＝x＋8 → 3x−6＝x＋8 → 2x＝14 → **x＝7**

検算: (1) 4×3−7＝5 ✓　(2) 左辺3×(7−2)＝15、右辺7＋8＝15 ✓

**Q6** 解とは「代入すると等式が成り立つ値」。x＝3を代入して
2×3＋a＝11 → 6＋a＝11 → **a＝5**

検算: 方程式2x＋5＝11をx＝3で確認: 6＋5＝11 ✓

### 段4 比例・反比例

**Q7**
(1) 比例だから y＝ax とおける。x＝4、y＝−12を代入: −12＝4a → a＝−3。**y＝−3x**
(2) y＝−3×(−2)＝**6**

検算: (1)の式でx＝4のときy＝−3×4＝−12 ✓

**Q8**
(1) **(イ) 半分になる**（xy＝12が一定だから、xが2倍になればyは1/2倍）
(2) y＝12÷3＝**4**

検算: (1)を具体で確認: x＝3→y＝4、x＝6（2倍）→y＝2（半分）✓　(2) 3×4＝12 ✓

### 段5 式の計算

**Q9**
(1) (5a＋2b)−(3a−4b)＝5a＋2b−3a＋4b＝**2a＋6b**
(2) 8a²b÷4a＝(8÷4)×(a²÷a)×b＝**2ab**（問題の注記どおり a≠0 が前提。a＝0では4aで割れない）

検算: (1)にa＝1、b＝1: もとの式(5＋2)−(3−4)＝7−(−1)＝8。答え2＋6＝8 ✓　(2)にa＝2、b＝3: もとの式8×4×3÷8＝12。答え2×2×3＝12 ✓
〔採点のポイント〕(1)で「−3a−4b」としたら、かっこを外すときの符号の処理でつまずいている。

**Q10**
(1) n＋(n＋1)＝**2n＋1**
(2) 模範解答例: 「2nはnがどんな整数でも偶数（ぐうすう）である。2n＋1は偶数より1大きい数だから、必ず奇数になる。」

検算: n＝3なら3＋4＝7（奇数）、2n＋1＝7 ✓。n＝−2なら−2＋(−1)＝−3（奇数）、2×(−2)＋1＝−3 ✓
〔採点のポイント〕(2)は次の2点が入っていれば正解: ①**2nが偶数**であること　②**偶数＋1だから奇数**という結論のつなぎ。「7とか9になるから」のような具体例だけの説明は、この段ではおしい。文字式は「どんな整数でも」を一気に言えるのが強みだからだ。

### 段6 連立方程式

**Q11** 2x＋y＝8 と x−y＝1 を辺どうし加えると（加減法）:
3x＝9 → x＝3。x−y＝1より y＝2。**x＝3、y＝2**

検算: 2×3＋2＝8 ✓、3−2＝1 ✓（**両方の式**で確かめるのが連立の検算の作法）

**Q12** りんごをx個、オレンジをy個とすると:
x＋y＝8（個数）
120x＋90y＝840（代金）
上の式から y＝8−x。代入して 120x＋90(8−x)＝840 → 120x＋720−90x＝840 → 30x＝120 → x＝4、y＝4。
**りんご4個、オレンジ4個**

検算: 個数4＋4＝8 ✓。代金120×4＋90×4＝480＋360＝840 ✓
〔採点のポイント〕答えの数値だけでなく、**2本の式を立てられたか**も見る。式なしで答えだけ当てた場合は、**正解に数えてよい**。このときの段6の正式な判定は「**通過（要注意フラグつき）**」とする（正解数の勘定は通過と同じで、要注意がフラグとして付く）。判定表の段6の調べるフレーズで立式を復習しておこう（中3の「方程式をつくる」で式を立てる力そのものが問われるため）。

### 段7 一次関数

**Q13**
(1) x＝1のとき y＝2×1−3＝−1。x＝4のとき y＝2×4−3＝5。yの増加量＝5−(−1)＝**6**
(2) 変化の割合＝yの増加量÷xの増加量＝6÷3＝**2**（一次関数ではxの係数と一致する）

検算: xの増加量は4−1＝3。6÷3＝2で、y＝2x−3の係数2と一致 ✓

**Q14** 傾き＝(9−5)÷(3−1)＝4÷2＝2。y＝2x＋bに(1, 5)を代入: 5＝2＋b → b＝3。
**y＝2x＋3**

検算: x＝1で y＝2＋3＝5 ✓、x＝3で y＝6＋3＝9 ✓（**2点とも**通ることを確かめる）

### 段8 図形

**Q15** 五角形の内角の和＝180°×(5−2)＝540°。正五角形は5つの内角がすべて等しいから
540°÷5＝**108°**

検算: 108°×5＝540°＝180°×3 ✓

**Q16**
(1) 180°−35°−80°＝**65°**
(2) 外角＝180°−65°＝**115°**

検算: 「三角形の外角は、となり合わない2つの内角の和に等しい」で確認: 35°＋80°＝115° ✓（(1)(2)を別々の方法で出して一致した。これはよい検算になっている）

**Q17** **(ア)・(イ)・(エ)**

〔採点のポイント〕(ウ)「2組の角がそれぞれ等しい」を選んだら誤り。これは合同条件ではない。角だけでは形が同じでも**大きさ**が決まらないからだ。実はこの(ウ)、中3で学ぶ**相似条件**の1つとして再登場する。選んでしまった人は、相似の単元でこの区別がそのまま主役になるので、覚えておくと得をする。

### 段9 データの活用・確率

**Q18** 相対度数＝その階級の度数÷度数の合計＝6÷40＝**0.15**

検算: 0.15×40＝6 ✓

**Q19**
(1) 公平な2枚の硬貨をたがいに影響なく投げるので、2枚の表裏の出方は（表・表）（表・裏）（裏・表）（裏・裏）の4通りで、どれも同様に確からしい。2枚とも表は1通りだから **1/4**
(2) **×**。確率1/4は「投げる回数を非常に多くしたとき、起こる割合が1/4に近づく」という意味であって、4回投げれば必ず1回起こることを保証するものではない（4回連続で起こらないことも、2回起こることもふつうにある）

〔採点のポイント〕(1)で1/3と答えた場合、（表・裏）と（裏・表）を1通りと数えている。2枚を区別して数え上げるのがポイント。

### 段10 中3への入口

**Q20** −3x(2x−5)＝−3x×2x＋(−3x)×(−5)＝**−6x²＋15x**

検算: x＝2を代入。もとの式: −6×(4−5)＝−6×(−1)＝6。答えの式: −24＋30＝6 ✓
〔採点のポイント〕「−6x²−15x」なら、負の数×負の数の符号処理でつまずいている（段1のQ1(3)とつながる）。

**Q21**
(1) **4と−4**（4²＝16、(−4)²＝16。2つとも書けて正解。Q2(2)と同じく「すべて」がカギ）
(2) **表せない**。理由の例: 「3²＝9、4²＝16 だから、2乗して10になる数は3と4の間にあり、整数ではない。」

検算: 3²＝9＜10＜16＝4² ✓
〔採点のポイント〕(2)は「3と4の間にはさまれる」という比較が入っていれば正解。この「整数で表せない長さ」に名前をつけるのが、平方根の単元の出発点になる。

**Q22**

| x | −2 | −1 | 0 | 1 | 2 |
|---|----|----|---|---|---|
| y | 4 | 1 | 0 | 1 | 4 |

検算: (−2)²＝4、(−1)²＝1、0²＝0 ✓（負の数の2乗が**正**になることに注意。(−2)²＝−4としたら段1の累乗の復習が効く）

## 判定表——ここから始めよう

### 手順

1. 段ごとの正解数を並べてみよう（例: 段1=2/2、段2=2/2、…、段8=1/3、…）
2. **段の過半数を誤った**（2問中2問誤り、3問中2問以上誤り）最初の段を探す。それが「停止候補」
3. その**次の段が通過でなければ**（過半数誤りの場合はもちろん、1問誤りの「要注意」の場合も）、停止候補の段が**あなたの学び直しの開始地点**。次の段が「要注意」のときも開始地点とするのは、安全側＝より手前から始める側に倒す暫定規則だ。次の段が通過していれば、停止候補の段は「その分野だけの穴」なので、下の表のその段の行だけ対応すればよい。なお、最後の段10には「次の段」がないので、段10で3問中2問以上誤ったら、そのまま停止として下の表の段10の行へ進もう。開始地点が決まったあとも、さらに後ろの段に停止候補（過半数誤り）があれば、開始地点は変えずに「**追加の穴**」として記録し、その段の行も読んで対応しよう。また、この「次の段」での確定は系統順の隣どうしで機械的に見る簡便法で、隣の段が別の分野（枝）のこともある。その場合も、安全側（より手前から始める）に倒れる設計にしてある
4. 段の1問だけの誤りは「要注意」——先へ進んでよいが、下の表の「調べるフレーズ」でその分野を軽く見直しておこう
5. **複数の枝で停止した場合**は、最も早い（番号の小さい）段の停止を主経路とし、その行き先単元のL01の直接の前提にあたる段（例: ④関数y=ax²に対する段4・段7）は「あわせて復習」として併せて対応しよう。過半数を誤った段の行は、主経路以外も**すべて読む**のが安全だ
6. **途中の段までで区切った場合**、そこまでの結果による判定は**暫定判定**だ。最後に解いた段が停止候補のときは、その次の段まで解いてから判定を確定しよう

### 大事な前置き——中1・中2の教材について

この判定表には「中1・中2の内容を学び直してから」という行き先が出てくる。**正直に書くと、中1・中2の教材はこのパッケージにはまだ存在しない**（今後整備予定）。当面は、手持ちの教科書・検索・AIチャットで橋渡ししてほしい。そのために各行に**調べるフレーズ**（そのまま検索窓に入れられる言葉）を付けてある。AIチャットを使うなら、たとえばこう頼むとよい:

> 「中1数学の『正負の数』を学び直したい中学3年生です。かっこのついた加減と、負の数の乗法の符号のルールを、例題3問つきで基礎から説明してください。」

- **複数の段で止まった**: いちばん早い（番号の小さい）段から埋めるのが原則。数学の系統は下の段が上の段を支えているからだ。ただし、枝がちがう場合（例: 段4の関数と段8の図形）はどちらから直してもよい。枝どうしは独立している
