# 解答集 L01〜L05

## L01 定理を見いだす——3つの正方形

**練習1** 囲む正方形は1辺6マスで面積 6×6＝36。四すみの直角三角形は1つ 2×4÷2＝4 で4つ分16。傾いた正方形の面積は 36−16＝**20**。
検算: 別の切り分け（マスの数え上げ）でも20になることを確認できる。

**練習2** 小さい2つの正方形は 2²＝4 と 4²＝16。和は 4＋16＝20 で、一番大きい正方形の面積20と一致。**成り立っている**。

**練習3** 予想: 16＋49＝**65**。直角をはさむ2辺は、16＝4²、49＝7² より **4マスと7マス**。
検算: 囲む正方形1辺11マス＝121、三角形4つ分 4×(4×7÷2)＝56、121−56＝65 ✓

**S1** 囲む正方形の面積は (a＋b)²。四すみの直角三角形は1つ ab/2 で4つ分 2ab。よって傾いた正方形の面積は
(a＋b)²−4×(ab/2)＝a²＋2ab＋b²−2ab＝**a²＋b²**
検算: a＝3、b＝4 を代入すると 9＋16＝25 で、L01本文の実測25と一致 ✓（傾いた正方形の1辺は斜辺だから、この式はそのまま「斜辺の上の正方形＝a²＋b²」、つまりL02の定理の証明のアイデアになっている。）

## L02 三平方の定理——意味・証明を知る・基本計算

**練習1**
(1) x²＝6²＋8²＝36＋64＝100 → x＝**10cm**。検算: 10²＝100 ✓
(2) x²＝1²＋1²＝2 → x＝**√2 cm**。検算: (√2)²＝2＝1＋1 ✓
(3) x²＝4²＋5²＝16＋25＝41 → x＝**√41 cm**（41は平方因数を持たないのでこれ以上簡単にならない）。検算: (√41)²＝41 ✓

**練習2** 斜辺が10cmだから x²＋6²＝10²。x²＝100−36＝64 → x＝**8cm**。検算: 8²＋6²＝64＋36＝100 ✓

**練習3** 斜辺が8cmだから x²＋5²＝8²。x²＝64−25＝39 → x＝**√39 cm**（39＝3×13で平方因数なし）。検算: 39＋25＝64 ✓

**練習4** x²＝7²＋(√15)²＝49＋15＝64 → x＝**8cm**。検算: 8²＝64 ✓（√を含む辺でも、2乗すれば普通の数として扱える例。）

**S1** △ABC（∠A＝90°）の頂点Aから斜辺BCに垂線AHを下ろすと、
- △ABCと△HBA: ∠Bが共通、∠BAC＝∠BHA＝90° → 2組の角がそれぞれ等しく相似（対応: A↔H、B↔B、C↔A）。
- △ABCと△HAC: ∠Cが共通、∠BAC＝∠AHC＝90° → 相似（対応: A↔H、B↔A、C↔C）。
よって **△ABC∽△HBA∽△HAC**。3つの相似な直角三角形が入れ子になっている、という見通しが立てば十分（この対応辺の比の式から定理を導くのが「相似による証明」の道すじ）。

**S2** x²＝5²＋12²＝25＋144＝169 → x＝**13cm**。3辺は5・12・13で**すべて整数** ✓ 検算: 169＝13² ✓（3辺が整数になる組はほかにも存在する。探索は発展の楽しみとして、本文の到達目標には含めない。）

## L03 三平方の定理の逆

判定手順（共通）: 最長辺をcとし、a²＋b²とc²を比べる。

**練習1**
(1) 最長10。6²＋8²＝36＋64＝100＝10² → **直角三角形といえる。斜辺は10cmの辺**。
(2) 最長8。5²＋6²＝25＋36＝61、8²＝64。61≠64 → **いえない**。
(3) 最長2√2（(2√2)²＝8＞4）。2²＋2²＝8＝(2√2)² → **いえる。斜辺は2√2cmの辺**。
(4) 最長4（4²＝16、(√11)²＝11、(√5)²＝5）。(√5)²＋(√11)²＝5＋11＝16＝4² → **いえる。斜辺は4cmの辺**。

**練習2** xの辺が斜辺だから x²＝9²＋12²＝81＋144＝225 → x＝**15**。検算: 15²＝225 ✓

**練習3** 解答例: 「6²＋8²＝100＝10² が成り立つので、**三平方の定理の逆**により、この三角形は10歩の辺を斜辺とする直角三角形になる。だから10歩の辺の向かい側の角、つまり6歩と8歩の辺にはさまれた角がちょうど直角になるから。」

**S1** 5・12・13: 5²＋12²＝25＋144＝169＝13² → **直角三角形**。
6・8・11: 6²＋8²＝100、11²＝121。100＜121 → 直角三角形ではない。図をかくと、斜辺候補11が「直角のときの長さ（10）」より長い分、向かいの角は開き気味になる——**90°より大きい（鈍角）と予想できる**。（予想まででよい。a²＋b²とc²の大小と角の大小の一般的な関係づけは高校で扱う。）

## L04 正三角形の高さと三角定規の3辺の比

**練習1** 高さhは、斜辺10cm・底辺の半分5cmの直角三角形で h²＝10²−5²＝100−25＝75 → h＝√75＝**5√3 cm**。検算: (5√3)²＋25＝75＋25＝100 ✓
面積＝10×5√3÷2＝**25√3 cm²**。

**練習2** 頂点から底辺への垂線は底辺を2等分する（二等辺三角形の性質）から、斜辺5cm・底辺4cmの直角三角形で h²＝5²−4²＝25−16＝9 → h＝**3cm**。検算: 9＋16＝25 ✓
面積＝8×3÷2＝**12cm²**。

**練習3** 1:1:√2 より斜辺＝7×√2＝**7√2 cm**。検算: 7²＋7²＝98＝(7√2)² ✓

**練習4** 1:2:√3 で、1にあたる辺が4cm。斜辺（2にあたる辺）＝**8cm**、残りの辺（√3にあたる辺）＝**4√3 cm**。検算: 4²＋(4√3)²＝16＋48＝64＝8² ✓

**練習5** 対角線は正方形を直角二等辺三角形2つに分ける。1:1:√2 より対角線＝6√2 → **6√2 cm**。検算: 36＋36＝72＝(6√2)² ✓

**S1** 高さ: h²＝a²−(a/2)²＝a²−a²/4＝3a²/4 → h＝ **（√3／2）a**。
面積＝a×(√3/2)a÷2＝ **（√3／4）a²**。
a＝6 の検算: h＝3√3、面積＝(√3/4)×36＝9√3 → 例題1と一致 ✓

**S2** 貼り合わせる辺（斜辺）の両端で、2枚の三角形の角が合わさる。45°定規の斜辺の両端の角は45°と45°、30°・60°定規の斜辺の両端の角は30°と60°。合わせ方は2通りあるが、いずれも4つの角は
**90°、90°、45°＋30°＝75°、45°＋60°＝105°**
検算: 90＋90＋75＋105＝360°（四角形の内角の和）✓

## L05 対角線・弦・2点間の距離

**練習1** 対角線²＝5²＋7²＝25＋49＝74 → **√74 cm**（74＝2×37で平方因数なし）。検算: (√74)²＝74 ✓

**練習2** 横をxとすると x²＋6²＝10²。x²＝100−36＝64 → **8cm**。検算: 64＋36＝100 ✓

**練習3** 弦の半分をxとすると x²＋4²＝6²。x²＝36−16＝20 → x＝2√5。弦＝2×2√5＝**4√5 cm**。検算: (2√5)²＋16＝20＋16＝36 ✓

**練習4** 弦の半分は8cm。距離dは d²＋8²＝10²。d²＝100−64＝36 → d＝**6cm**。検算: 36＋64＝100 ✓

**練習5** 横のずれ 5−2＝3、縦のずれ 5−1＝4。AB²＝3²＋4²＝25 → AB＝**5**。検算: 9＋16＝25 ✓

**練習6** 横のずれ 1−(−3)＝4、縦のずれ −1−2＝−3 →長さとしては3。CD²＝4²＋3²＝25 → CD＝**5**。検算: 16＋9＝25 ✓

**S1** 各辺:
- OP²＝4²＋2²＝20 → OP＝2√5
- OQ²＝2²＋6²＝40 → OQ＝2√10
- PQ²＝(4−2)²＋(2−6)²＝4＋16＝20 → PQ＝2√5

OP＝PQ＝2√5 だから**二等辺三角形**。さらに最長辺はOQで、OP²＋PQ²＝20＋20＝40＝OQ² が成り立つから、三平方の定理の逆により**∠Pを直角とする直角三角形**。あわせて**直角二等辺三角形**である。
検算: 直角二等辺なら斜辺＝他の辺×√2 のはず → 2√5×√2＝2√10＝OQ ✓
