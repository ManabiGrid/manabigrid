# セルフQAチェックリスト（工程4用）v1

執筆が終わったドラフトを、**執筆とは別の会話（新規チャット）で**検査するためのチェックリストです。工程全体での位置づけは [../OPS.md](../OPS.md) を参照してください。

検査の約束:

- 検査担当は**書き直しをしない**。指摘だけを返す（修正は執筆側の仕事——作る者と検査する者の分離）
- 入力: 教材ドラフト一式＋執筆用サマリー（writing_summary.md）＋このチェックリスト
- 各指摘は「**箇所・違反した規則・修正案**」の3点セットで書く
- 指摘がない項目は「なし」と明言する（書かずに済ませない）

## 点検項目

### ① 数値の年度なし使用ゼロ

- [ ] すべての統計値（正答率・反応率等）に年度・出典が付いているか

### ② サマリー外の断定ゼロ

- [ ] 執筆用サマリーにない事実の断定がないか（verify_required 注記のない追加事実は違反）

### ③ 解答漏れゼロ

- [ ] 本文に解答が混在していないか（解答は別ファイルのはず。レッスン末に解答ファイルへのリンクがあるか）

### ④ レベル外混入ゼロ

- [ ] 対象読者レベル外の内容（入試技巧・先取り）が標準本文に混ざっていないか（発展は stretch に分かれているか）

### ⑤ 範囲外への踏み込みゼロ

- [ ] この単元で「扱わない」と決めた範囲に踏み込んでいないか

### ⑥ 数値解答の再計算（最重要・実行的検証）

- [ ] 全問を実際に解き直し、解答ファイルと一致するか確認したか（見た目の確認でなく再計算）

### ⑦ 他者前提活動ゼロ（学習者単独成立）

- [ ] 他者の協力を前提にした活動（ペア活動・発表・インタビュー・家族への協力依頼等）が本文にないか。独習で完結するか

### ⑧ 指標名照合と合計再計算

- [ ] 数値の指標名を出典と照合したか（正答率と反応率の取り違え等）。数値表はサマリーの**全項目と全数照合**する（存在する行だけの照合では欠落を検出できない）
- [ ] 検査の根拠に使う数値自体にも出典があるか（出典なしの記憶値で検査しない）
- [ ] 時間配当の合計再計算: lesson_map の節小計・合計・「◯時間版」の記述が算術的に一致するか

### ⑨ 文体点検

- [ ] 「〜してみて！」形がゼロか（基本形は「〜してみよう」）
- [ ] ——（罫線ダッシュ）が1レッスン1〜2回以内か。「そう、」「実は」の乱発・比喩の連打・体言止めの連続がないか
- [ ] 難読語・固有名詞の初出に読み（ふりがな）が付いているか
- [ ] 手書きと差異が出る記法に「ノートではこう書く」の橋渡しがあるか

### ⑩ 中心例の意味検算

- [ ] 単元の中心例・中心主張について、「式・結論が主張する意味」を数値反例と依存関係チェックで検算したか。例:「pは答えに関係ない」型の主張は、pを実際に動かして結論の真偽を確認する（値の一致検算だけではこの層の誤りは見つからない）

### ⑪ 量化・定義域・包含の検査

- [ ] 「どの〜でも」「必ず」「1つ分かれば決まる」型の全称・存在主張に、0・境界値・退化ケース（分母0・原点等）の反例チェックを行ったか
- [ ] 概念の包含関係（例: 比例は一次関数の特別な場合）を取り違えていないか

### ⑫ 根拠区分ラベルの妥当性

- [ ] 教授法判断の根拠区分ラベル（Established / Adapted / Derived / Experimental）に漏れがないか（teacher_notes・guide層の主要な設計判断が対象）
- [ ] `Established` を名乗る判断に根拠参照（研究・実践の出典）が付いているか

## 検査後

- 指摘は執筆の会話に戻して修正する（検査担当が直接修正しない）
- 指摘ゼロでも「全項目を点検して指摘なし」と明言して終える
