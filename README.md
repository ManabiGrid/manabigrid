# ManabiGrid（まなびグリッド）ベータ版

![まなびグリッド — 無料で・一人で・つまずいたところから学び直せる。AIと人間の共同制作によるオープン教材プロジェクト](docs/assets/banner.png)

> 📖 **勉強しに来た中学生・高校生のみなさんへ**: あなた向けの案内は **[はじめての人へ（FOR_STUDENTS.md）](FOR_STUDENTS.md)** にあります。
>
> このページは、プロジェクトを手伝ってくれる大人・開発者・AI向けの説明です。

**どんな環境の子でも、無料で・一人で・つまずいたところから学び直せる**——そのための中高5教科の教材を作って配る、オープン教材プロジェクトです。

本文はAI「Claude Fable 5」が多段階工程で執筆し、別ベンダのAIによる批判レビューを経て、人間（現役の個別指導塾講師）が通読・検収したうえで公開しています。ただし、**単元別の正式な「人間レビュー済」状態への昇格は全件これから**です（各単元の到達状態は [進捗一覧](curriculum/PROGRESS_INDEX.md)）。

**このプロジェクトはまだ作りかけです。そして、これからも作り続けます。** 完成を待ってから公開するのではなく、検証済みの単元から順に公開しながら育てる方針です。

名称の「ベータ版」は、教材がまだ育ちきっていないことの正直な表示です。**目安として、中学数学（中1〜中3）の全単元が人間レビュー済みでそろった時点で「ベータ版」を外します**（基準は状況に応じて見直し、変更時はこのREADMEに記録します）。

---

## なぜ作るのか

教科書も参考書もよくできています。それでも「一度つまずいた子が、誰にも質問できない環境で、お金をかけずに、戻ってやり直せる教材」はほとんどありません。

つまずいたところまで戻って、自分のペースでやり直せる環境を、だれでも・無料で使えるかたちにする——それがこのプロジェクトの目的です。

## どう作っているか（Claude Fable 5 と人間の共同制作）

本文は **Claude Fable 5**（Anthropic の最上位クラスのAIモデル）が多段階の工程（調査→正規化→執筆→セルフQA）で執筆し、別ベンダの **GPT-5.6 Pro**（OpenAI）による批判レビューを経て、人間（現役の個別指導塾講師）が通読・検収したうえで公開しています。ただし、**単元別の正式な「人間レビュー済」状態への昇格は全件これから**です（各単元の到達状態は [進捗一覧](curriculum/PROGRESS_INDEX.md)）。

品質を守る仕組みの多く——図をコードから生成して数値を自動検算する方式、解答の数値が図に漏れていないかの機械検査、1つの原稿から生徒用・自習用・教師用を厳密一致で生成する仕組み——は、制作の途中で Fable 5 自身が設計したものです。

各工程のプロンプトはそのまま公開しています。**同じ手順を、あなたの Fable 5 で再現できます**（記録: [docs/METHODOLOGY.md](docs/METHODOLOGY.md)／プロンプト: [docs/prompts/](docs/prompts/)）。

きっかけの一つは、漫画『チ。―地球の運動について―』——知が環境を越えて受け継がれていく熱を、いまの道具でやる実験です。

※本プロジェクトは Anthropic・OpenAI と提携・公認関係にありません。モデル名は使用ツールの事実表示です。

## 参加のしかた（貢献の案内）

くわしくは **[CONTRIBUTING.md](CONTRIBUTING.md)（参加ガイド）** へ。型は7つあります:

| 型 | 概要 | 使うAI・道具 |
|---|---|---|
| 誤り報告 | 誤り・分かりにくさをIssueかフォームで。いちばん助かります | 何でもOK（AIなしでも） |
| 素材・知見の提供 | 誤答パターン・つまずき事例・効いた説明（経験談歓迎） | 何でもOK |
| 教材の改善 | 説明の改善・弱点指摘のIssue/PR（提出物は入力素材——採用時はメンテナ側の検証工程を通して反映） | どのAIでもOK |
| 新単元の制作 | 公開プロンプトで1単元を直列に仕上げてPR | **Claude Fable 5 限定**（Pro反証はメンテナ側で実施・理科のみ例外あり→ [docs/MODEL_POLICY.md](docs/MODEL_POLICY.md) §5） |
| デザイン | 図版・レイアウト・読みやすさの提案〜実装 | 指定なし |
| 仕組みへの提案 | 工程・検査・構造の改善（【設計】Issueで議論から） | 指定なし |
| 広める | SNSでの紹介・友達や同僚に教える・学校や塾で話題にする | 何もいりません |

- **広めることも立派な貢献です**——このプロジェクトは知られるほど、教材が必要な子に届き、誤り報告や改善の目が増え、教材の質が上がります。リンクをひとつ紹介してもらえるだけでありがたいです
- **良かった点の報告も、誤り報告と同じくらい歓迎です**——「どの説明が効いたか」が分かると、その良さを他の単元へ広げられます（[おたよりフォーム](https://docs.google.com/forms/d/e/1FAIpQLSeKBOQ9EB_DGfkh7Y0LYU83w6ghkPTQfH65BSFjVVWtwEEt9Q/viewform)は匿名OK・アカウント不要）

- いちばん簡単な参加: あなたの Claude Code（Fable 5）にこのリポジトリのURLを渡して「参加したい。進めておいて」と言うだけです（[CLAUDE.md](CLAUDE.md) が案内します。Fable 5 以外のAIには [AGENTS.md](AGENTS.md)）。行動規範は [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

- 教材の中に「教科書や過去問と似すぎている部分」など、他者の権利にふれていそうな箇所を見つけた場合も、同じ窓口（[Issue](../../issues)・おたよりフォーム）で教えてください。確認のうえ、すみやかに対応します
